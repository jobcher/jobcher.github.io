Linux 6.18 正式确认为 2025 年 LTS 内核

Linux 6.18 内核已被官方提升为今年的长期支持（LTS）版本。Linux 稳定版维护负责人 Greg Kroah-Hartman 已正式确认该版本将进入长期维护周期，并在 kernel.org 的发布页面完成标注，Sasha Levin 同样参与维护。

当前规划显示，Linux 6.18 LTS 将至少支持至 2027 年 12 月，约 24 个月。维护周期是否延长，将取决于企业采用情况、测试反馈以及补丁回溯的需求规模。

每年的 LTS 版本通常由当年度最后一个主线版本担任，因此 6.18 成为 LTS 并不意外，但官方确认意味着该版本的长期维护路径已经确定。与此同时，Linux 6.19 稳定版预计将于明年 2 月发布。

Steam Linux 占比再创新高：11 月升至 3.2%

WebGPU 正式在主流浏览器启用 WebGPU 已在 Chrome、Edge、Firefox 与 Safari 上获得正式支持，标志着网页原生高性能图形与 GPU 通用计算能力的全面落地。这一里…

WebGPU 正式在主流浏览器启用

WebGPU 已在 Chrome、Edge、Firefox 与 Safari 上获得正式支持，标志着网页原生高性能图形与 GPU 通用计算能力的全面落地。这一里程碑来自 W3C GPU for the Web 工作组多年的协作，参与方包括 Apple、Google、Intel、Microsoft 与 Mozilla。

WebGPU 提供较 WebGL 更现代、性能更高的接口，允许浏览器直接利用当代 GPU 特性，用于高端 3D 图形渲染与通用计算任务。其计算管线进一步提升机器学习推理、视频处理、物理仿真等重型工作负载的效率，使网页端获得接近桌面级的性能。

多项 AI 与图形框架已基于 WebGPU 实现加速，包括 ONNX Runtime、Transformers.js、Babylon.js、Three.js 等。借助 Render Bundles 等新机制，渲染场景的 CPU 负载可显著降低，性能提升可达数量级。

在平台覆盖方面，WebGPU 已在 Windows、macOS、ChromeOS、iOS、iPadOS、visionOS，以及部分 Android 设备上线；Linux 与更多平台的支持仍在推进中。

美国对加勒比海地区所谓“运毒船”的二次打击引发争议。
美军9月2日对一艘“运毒船”进行导弹打击。首次袭击后，据报有2名生还者。美军又进行了第二次打击，令2名生还者死亡。但“二次打击”的做法违反了美国法律。
消息人士称，国防部长赫格塞思在首次打击前口头下令，要求“干掉船上每个人”。在发现生还者后，美军特种作战司令部司令弗兰克·布拉德利下令再次打击，以遵守赫格塞思的命令。
国会两个由共和党领导的委员会已针对此次袭击启动两党事实调查。参议院军事委员会主席罗杰·威克称，他已和赫格塞思等人交谈，并将与布拉德利会面，正寻求打击行动的音视频，随后采取下一步行动。
在12月2日的内阁会议上，赫格塞思称其过了“几个小时”才知道需要额外打击来杀死两名生还者。他说自己没见过生还者，船只爆炸的火和烟导致什么都看不见，这就叫“战争迷雾”。
特朗普也表示自己并不知道“二次打击”的事情，自己并未参与此事。但五角大楼早些时候表示，所有空袭均由总统指挥。布拉德利尚未公开置评。
知情人士称，美军在策划后续任务时，更加重视营救那些袭击中的生还者。尚不清楚是谁进行了这一变更。
（华盛顿邮报，美联社）
