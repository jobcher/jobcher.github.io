🖼 从助教到内蒙古主席 61岁女高官王莉霞落马

从助教到内蒙古主席 61岁女高官王莉霞落马

来自蒙古族，女性，拥有经济学博士学位、教授职称，内蒙古自治区政府主席王莉霞的上述因素曾闪耀其履历，助推其仕途，但这一切均在近期划上休止符。8月22日，中央纪委国家监委网站发布消息称，内蒙古自治区党委副书记、自治区政府主席王莉霞涉嫌严重违纪违法，目前正接受中央纪委国家监委纪律审查和监察调查。

被查消息公布前六天，王莉霞还曾公开露面。据《内蒙古日报》8月17日报道，8月16日，巴彦淖尔乌拉特后旗发生山洪导致人员失联。按照内蒙古党委安排，王莉霞紧急赶赴现场，指挥人员搜救；随后主持召开视频调度会，传达国务院有关领导批示要求，通报各盟市风险点位和重点部位，要求按照内蒙古党委和政府关于防汛工作的部署，进一步细化防范举措，全力做好防汛救灾工作。此外她还到医院看望了获救人员。

至此，王莉霞成为2025年继金湘军、蓝天立之后第三名被查的在任省级政府一把手，还是今年继齐扎拉、蒋超良、金湘军、蓝天立、毕井泉、刘慧之后被查的第七名正省部级高官，以及中共二十大以来继李微微、赵金云、刘慧之后被查的第四名女高官。

今年4月15日，中央纪委国家监委发布消息称，中央第十巡视组巡视内蒙古工作动员会召开，中央巡视组将在内蒙古工作2个半月左右。巡视组受理信访时间截止到2025年6月23日。根据规定，中央巡视组主要受理反映内蒙古省级领导班子及其成员、下一级党组织领导班子主要负责人和重点岗位领导干部问题的来信来电来访，重点是关于违反政治纪律、组织纪律、廉洁纪律、群众纪律、工作纪律、生活纪律等方面的举报和反映。

消息来源: 财新

特斯拉将在中国集成Deepseek、豆包AI语音控制功能

根据特斯拉公司官网上的服务条款文件，该公司计划推出由Deepseek和字节跳动旗下豆包AI驱动的车载语音助手功能，以与本土竞争对手展开竞争。该系统将让驾驶员通过语音指令控制导航、娱乐和车厢温度，并接收新闻更新，其中豆包负责处理大部分功能，Deepseek负责管理AI交互，两者均托管在字节跳动的火山引擎云平台上。目前尚不清楚这些模型是否已经集成到特斯拉的中国版汽车中，因为多位车主告诉彭博新闻，他们尚未收到相关通知，而特斯拉官方微信应用程序上的最新空中升级更新日期为8月18日。特斯拉和字节跳动的代表均未回应置评请求。

LibreOffice 25.8 发布

LibreOffice 25.8 发布

The Document Foundation 宣布发布 LibreOffice 25.8。主要新变化包括：增强了用户界面，Welcome/What’s New 对话框能直接访问用户界面选择器和外观选项；改进了打开文档的速度和滚动性能，Writer 和 Calc 打开文档的速度提升了最多 30%；优化内存管理，虚拟桌面和瘦客户端上的操作更流畅；增强了与 Microsoft Office 文件格式的互操作性，能更精确处理 DOCX、XLSX 和 PPTX 文件，减少格式问题；支持导出 PDF 2.0 格式，等等。
