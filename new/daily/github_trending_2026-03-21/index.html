Cursor 推出 Composer 2

Cursor 推出 Composer 2

3 月 19 日，Cursor 宣布在产品中上线 Composer 2，将其定位为新一代代码生成与软件工程模型。根据官方介绍，Composer 2 的定价为每百万输入 token 0.50 美元、每百万输出 token 2.50 美元，Cursor 同时提供一个速度更快的变体，价格为每百万输入 token 1.50 美元、每百万输出 token 7.50 美元，并计划将快速版设为默认选项。

Cursor 公布的基准结果显示，Composer 2 在 CursorBench、Terminal-Bench 2.0 和 SWE-bench Multilingual 上分别取得 61.3、61.7 和 73.7 分，均高于 Composer 1.5 的 44.2、47.9、65.9，以及 Composer 1 的 38.0、40.0、56.9。Cursor 表示，这一提升来自其首次持续预训练运行，以及面向长周期编程任务的强化学习训练。

Firefox 将上线内置免费 VPN

Firefox 将上线内置免费 VPN

Mozilla 于 2026 年 3 月 17 日宣布，Firefox 149 将从 3 月 24 日起加入免费内置 VPN。按官方表述，这项功能会将 Firefox 的浏览流量经代理转发，以隐藏用户的 IP 地址和位置，无需额外下载独立应用。首批开放地区为美国、英国、德国和法国，每月提供 50GB 流量。这项免费服务将建立在自身的数据原则之上，并试图与常见的数据处理不透明的免费 VPN 服务拉开距离。此外，Firefox的VPN服务长期和mullvad保持合作。

OpenAI 收购 Astral，将 uv、Ruff、ty 纳入 Codex

OpenAI 于 2026 年 3 月 19 日宣布，计划收购开源 Python 工具公司 Astral。Astral 是 uv、Ruff 和 ty 的开发方，这三款工具分别覆盖 Python 包与项目管理、代码检查与格式化，以及类型检查与语言服务。OpenAI 表示，交易完成后，Astral 团队将加入 Codex 团队，并探索与 Codex 的更深层集成。
