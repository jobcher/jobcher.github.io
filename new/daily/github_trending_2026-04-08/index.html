🖼 智谱发布 GLM-5.1：开源 Agent 智能体编程模型性能登顶

智谱发布 GLM-5.1：开源 Agent 智能体编程模型性能登顶

智谱正式推出新一代旗舰级 Agent 工程模型 GLM-5.1，并宣布遵循 MIT 协议开源。该模型在 SWE-Bench Pro 等核心编程基准测试中位居开源首位、全球第三，展现出超越前作的复杂任务处理能力。GLM-5.1 专为长时程任务设计，支持长达 8 小时的自主运行，能够通过数千次工具调用与持续策略优化，在数据库优化及 GPU 内核开发等场景中实现显著的性能突破。
目前，GLM-5.1 的模型权重已在 HuggingFace 及 ModelScope 同步上线，开发者可通过 api.z.ai 或 BigModel.cn 进行接入。该模型同时适配 Claude Code 和 OpenClaw 等主流编程框架，为构建高效率的自主 AI 程序员提供了底层支持。

来源：z.ai

由于在工程验证测试阶段遇到超出预期的技术难题，苹果公司首款折叠屏iPhone的量产及出货计划可能面临数月的延迟。消息人士透露，今年4月至5月是解决相关工程挑战的关键期。尽管苹果已调整策略优先保障高端机型生产，但目前的工程方案尚不足以完全克服折叠屏特有的技术障碍，部分零部件供应商已收到生产计划可能推迟的通知。
按原计划，苹果首批折叠屏iPhone预计产量约700万至800万部，占新机总量的10%以下。市场分析认为，苹果的入局对推动折叠屏手机市场至关重要，IDC预测2026年全球折叠屏出货量将受此带动增长约30%。目前，苹果及其供应链正全力应对技术攻关，以期降低对整体营销策略和上市节奏的影响。

海峡沿岸的社区报告称，在我们抵达前的两周左右，每天大约有两到四艘船只通过该航道。而我们在 4 月 2 日确认了 15 艘船只穿越海峡。虽然计数方法并不算机构级（一个视野良好的吧台凳，一部变焦到最大的国产手机，辅以水面上的笔记本记录），但它产生的数据却意义重大。

知情人士告诉我们，这种节奏在 4 日仍在持续，又有 15 至 18 艘船只通过。这相当于在两天内完成了此前一周的交通量。

确认通过的船只包括来自印度、马来西亚、日本、希腊、法国、阿曼、土耳其和中国的船只。中国船只被发现关闭了自动识别系统（AIS），正穿行于拉拉克-格什姆（Larak-Qeshm）航道。我们还目睹了似乎是首次完全在格什姆-拉拉克航道之外进行的航行：超大型油轮（VLCC）和液化天然气（LNG）运输船（空载）紧贴阿曼海岸行驶，独立于伊朗检查站进行过境。
