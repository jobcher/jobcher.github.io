Telegram 宣布视频功能升级及机器人消息限制和广告收入分成更新

1, 自适应视频播放机制：2024年10月31日，Telegram 宣布引入新的自适应视频播放功能。对于大频道中新上传的视频，播放器将根据用户的网络连接自动选择适当的质量等级，以优化数据使用并加快加载速度。用户还可以手动选择高、中或低质量。借助这一更新，大频道所有者可以上传更高质量的视频，而无需为观看体验进行预处理。Telegram 将为每个视频优化并压缩成多种分辨率。
注：此功能目前仅适用于拥有数千名订阅者以上的频道。

2,视频播放速度控制：用户可以通过新的手势快捷方式无缝调整视频播放速度，无需进入设置。按住视频右侧可激活1.5倍速，并滑动至2.5倍速，松开手指后恢复正常速度。在 Android 上，用户可以双击视频左侧或右侧分别跳过10秒。

3, 为已发送消息添加媒体：用户现可轻松为已发送的消息附加媒体，或编辑已发送的媒体来重新编辑照片或更换附件。

4, 最后编辑时间戳：编辑过的消息将显示最后一次编辑的时间戳。若多次编辑，则会显示最近的编辑时间。小型群组和私聊中还会显示详细的阅读回执，指明每个成员查看消息的具体时间。

5, 聊天专属标签：支持使用新的格式 #hashtag@username，点击后仅显示特定聊天内的相关内容和故事。

6, 机器人广告收入分成：Telegram Ads 现可显示在机器人和小程序中，这些服务可通过广告收入分成来支持开发。Telegram Ads 低调且重视隐私，与内容创作者及机器人开发者分享50%的广告收益。

7, 提升的机器人消息限额：所有 Telegram 机器人每秒可免费发送最多30条消息。为更好支持大规模服务，机器人可以启用增加的消息限额，每秒发送高达1000条，费用为每条消息0.1星，使用 Telegram 星支付。

8, 精确反应计数：用户可以长按反应计数器，查看帖子收到的具体反应数量，如心形、星星或其他表情符号。

经济学人：卡马拉·哈里斯的胜算为何突然上升了

有三个原因促成了这种变化。

其一是新民调数量的增加——今天模型中增加了65项调查数据，让模型对微小变化的判断更为自信，模型对预测的“确定性”更高。

随着新增民调数据的累积，模型能够更准确地识别出小幅的支持率变化，而不会将这些小变动视为统计误差或偶然波动。

其二是选举日期将至。迄今为止，模型一直在进行预测，候选人还有数周甚至数月的时间争取选民。而目前许多民调机构正在发布本选季的最后一轮调查，因此预测将很快变成一种“现时预测”。

第三个原因是选情异常接近，意味着即使是微小的支持率变化也能显著影响胜率概率。

昨天最具影响力的民调主要集中在四个州：密歇根州、北卡罗来纳州、宾夕法尼亚州和威斯康星州。

在这些州中，哈里斯的预期得票率平均上升了0.4个百分点，虽是小幅波动，但足以使她在这四个州的胜率平均上升六个百分点。
