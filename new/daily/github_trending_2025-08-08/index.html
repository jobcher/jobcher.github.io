中国7月出口激增7.2%，尽管美国加征关税，供应链转移加剧

根据周四公布的海关数据，中国7月份出口同比增长7.2%，意外加速，创下自4月以来的最快增速，高于经济学家预测的5.6%。进口增长4.1%，贸易顺差达到982亿美元。
尽管面临美国高额关税，中国出口仍表现出强劲韧性，表明全球需求仍在支撑中国经济。今年上半年，为规避关税，企业加快出货，推动出口创下纪录。不过，这一势头能否持续仍不确定，因为“抢出口”效应可能会逐渐减弱。
最新数据显示贸易活动有所放缓，截至8月3日的一周，中国港口处理的集装箱数量连续第二周下降。
在制造业方面，中国日益依赖越南、墨西哥等第三国完成终端产品或零部件的生产。这一趋势自首次中美贸易战及其后续限制措施加剧后明显加快。通过这些国家出口至美国的产品中，中国所占的增值制造比重已从2017年的14%上升至2023年的22%。
