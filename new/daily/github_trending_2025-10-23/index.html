Anthropic 推出 Claude Code 网页版：云端多任务编程助手上线

Anthropic 推出 Claude Code 网页版：云端多任务编程助手上线

2025年10月21日，Anthropic 宣布推出 Claude Code on the web，让开发者可直接在浏览器中分配和执行代码任务，目前以研究预览版（Research Preview）形式向 Pro 与 Max 用户开放。

这一版本让用户无需打开终端即可启动多项编程任务。通过连接 GitHub 仓库并描述任务需求，Claude 可在 Anthropic 托管的云端环境中自动实现代码修改、测试与 Pull Request 生成。所有任务在独立沙盒中运行，并支持实时进度跟踪与交互调整。

Claude Code 网页版支持并行运行多个项目任务，适合用于常规修复、后端改动和测试驱动开发等工作场景。Anthropic 同时推出移动端预览版，使开发者可在 iOS 设备上随时管理代码任务。

在安全性方面，Claude Code 采用隔离的沙盒执行机制与受限网络访问，所有 Git 操作均通过安全代理完成，确保 Claude 仅能访问授权仓库。开发者可自定义网络白名单，例如允许 Claude 下载 npm 依赖以执行测试。

OpenAI 推出 ChatGPT Atlas：内置 ChatGPT 的浏览器

2025年10月21日，OpenAI 宣布推出全新产品 ChatGPT Atlas —— 一款将 ChatGPT 深度集成的网络浏览器，率先登陆 macOS 平台。

Atlas 将搜索、浏览与 ChatGPT 的对话功能合为一体，使 AI 能直接理解用户在网页上的操作并执行任务，无需切换窗口或复制内容。用户可在浏览网页时启用 “agent 模式”，让 ChatGPT 自动执行如查找资料、预订服务、整理信息等操作。

Atlas 还引入了“浏览记忆”（browser memories），可让 ChatGPT 记住用户浏览过的内容，在未来的对话中提供更有上下文的帮助。该功能完全可选，用户可随时查看、删除或关闭相关数据。

安全方面，OpenAI 表示 Atlas 中的 ChatGPT 无法执行代码、下载文件或访问本地应用，并在涉及金融等敏感网站时自动暂停操作。同时，用户可在“隐身模式”下使用，以避免产生任何浏览记录或记忆。

目前，ChatGPT Atlas 已向 Free、Plus、Pro 和 Go 用户开放下载，Business 用户可申请测试版。Windows、iOS 和 Android 版本将随后推出。

来源： OpenAI 官方博客
