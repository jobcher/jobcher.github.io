日本厚劳省于4日发布人口动态统计数据，2024年日本出生人数为686,061人，首次低于70万人，创下自1899年有统计以来的最低纪录。总出生率为1.15，亦为战后的最低水平，少子化趋势加速。出生人数较去年减少41227人，降幅达5.7%。2022年出生人数刚刚跌破80万人，预计2024年出生人数为75万人，但实际人数已提前15年降至68万人。生育率呈现西高东低的倾向，最高在冲绳（1.54），最低在东京（0.96）。
（朝日新闻）

越南6月3日修改《人口法令》，取消“一孩或二孩”生育规定。

越南6月3日修改《人口法令》，取消“一孩或二孩”生育规定。
原规定“决定生育时间和间隔”被修改为：“个人或夫妻在平等基础上，根据年龄、健康状况、学习、工作、收入以及抚养子女的能力，自主决定生育时间、子女数量及每胎间隔时间”。同时，删除原第2款“每对夫妻只生育一个或两个孩子，特殊情况由政府另行规定”。
此次修改旨在解决各地区、各群体间生育率差异过大的问题，防止出生率过低、无法达到更替水平的情况出现。这可能对今后的经济、社会、国防和安全的可持续发展产生负面影响。
（越通社）
