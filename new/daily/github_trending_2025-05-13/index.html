谷歌十年来首次更新“G”图标 色彩渐变更显活力

谷歌近日对其标志性的“G”图标进行了十年来的首次更新。此次调整打破了原有红、黄、绿、蓝四色块的固定分区设计，改为色彩渐变交融的效果——红色渗入黄色，黄色过渡到绿色，绿色再融入蓝色，整体视觉更为鲜明动感。这一设计既呼应了Gemini的渐变风格，也与搜索功能的AI模式快捷图标相契合。目前，新图标已随昨日更新登陆iOS版Google搜索应用，Android用户亦可通过测试版（16.18）获取。由于改动细微，主屏幕或浏览器标签页上的小尺寸图标可能不易被立刻察觉。值得注意的是，谷歌此次并未同步更新六字母主标识，其他产品（如Chrome或Maps）的图标是否会跟进渐变设计尚不明确。

墙中之墙：中国地区性审查的兴起

墙中之墙：中国地区性审查的兴起

长期以来，中国的互联网审查有着相对集中的政策和统一的实现，这套系统被称为中国防火长城（GFW）。然而，自2023年8月以来，有传闻称河南省部署了自己的地区性审查系统。在这项工作中，我们对河南省的省级审查进行描述和分析，并将其与国家级的GFW进行了比较。我们发现，河南建立了基于TLS SNI和HTTP Host的审查机制，用于检测和封锁离开该省的流量。

虽然河南防火墙在复杂性和应对网络流量多样性方面有所欠缺，但其不稳定且激进的二级域名封锁策略，一度使其封锁的网站数量达到GFW的十倍之多。我们基于对河南防火墙的流量解析缺陷和注入行为的观察，提出了一些简单的仅需客户端实现的办法来绕过河南省的审查。我们的工作记录了一种值得警惕的现象，即中国的地区性审查正在抬头。
