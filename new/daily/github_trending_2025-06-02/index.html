然而，威特科夫称哈马斯的回应“完全不可接受”。他敦促哈马斯接受框架提议作为谈判的基础，并表示谈判可能在下周开始。以色列亦将哈马斯的回应视为“有效拒绝”威特科夫的提议，而是提出了由自己条件的新提议。
哈马斯政治局成员巴塞姆·纳伊姆质问为什么以色列可以不同意哈马斯与美国商定的条款，以色列的回应被作为谈判的唯一回应。他表示这违反了调解的完整性和公平性，构成对另一方的完全偏向。
在加沙，饥饿引发的混乱还在继续。联合国的援助车队在汗尤尼斯被临时路障拦下，随即被巴勒斯坦人抢走。除此之外，在5月底的3天内发生了4起联合国设施被抢的事件。而以色利和美国支持的“加沙人道主义基金会”在5月31日仅分发了30卡车的食物。
以色列还继续在加沙发动军事行动，加沙卫生部门称，过去24小时有至少60人死于袭击。以色列军方称，来自加沙的多枚炮弹落在了开阔地带。
（CNN，美联社）

5月23日，朝鲜驱逐舰下水事故调查组经调查，认定有关舰船底未发现破洞，船体右舷被刮削，船尾部份进水。舰船损毁程度较初期评估轻。后续排水与扶正工作需时2～3天，修复舷侧需10余天时间。 朝党中央军委指示：需要查明事故发生原因和责任人；无论舰船状态如何，要借此事故给在所有部门蔓延的缺乏警惕、不负责任和非科学的经验主义态度以严厉打击，敲响警钟。 朝鲜司法机关已于22日传拘了清津造船厂厂长洪吉浩。 （朝中社）

5月29日卫星图像显示，清津厂已在发生事故的崔贤级2号舰两侧部署30余个疑似气球，用以托举船体或防止其进一步下沉，此外，有数根缆绳与舰体相连，以稳定其位置，防止发生二次倾覆。与此同时，附近两艘支援船正在对舰体实施排水作业，以使舰艏能够从滑道装置上脱离并入水，便于后续开展扶正工作。
此外，为保障该舰日后顺利离港，港池内通往外海的唯一河道正进行清淤作业。
