Anthropic 收紧 Claude 订阅第三方使用

Anthropic Claude Code 团队成员 Thariq 表示，团队已加强对“冒充/伪装 Claude Code harness”的防护；此前有账户因第三方 harness 使用 Claude 订阅触发滥用过滤而被封禁。他称第三方 harness 使用 Claude 订阅会制造异常流量与排障困难，并违反其服务条款，建议在自建工具中使用官方 API 路径。

前端开源独角兽 Tailwind Labs 近日陷入生存危机。联合创始人 Adam Wathan 证实，受 AI 大模型冲击，公司营收自 2023 年以来锐减近 80%，官方文档流量下跌 40%。为保生存，公司已裁撤工程团队 75% 的人员。危机在一次 GitHub PR 争议中曝光。社区提议增加 llms.txt 以优化 AI 读取体验，被 Wathan 拒绝。他直言，当前优先级是“让业务可持续并能支付账单”；他称让 LLM 更容易读取官方文档会进一步降低文档访问，从而减少用户发现其付费产品的机会。

Business Insider 报道称，所谓 75% 对应“4 名工程师裁 3 人”；裁员后团队规模约为三位创始人、1 名工程师及 1 名兼职人员。报道同时提到，Wathan 表示若不改变现状，公司可能在约 6 个月内无法满足发薪需求，因此选择在尚能提供遣散补偿时进行裁员。

事件发酵后，部分公司公开表达赞助支持。Vercel CEO Guillermo Rauch 表示 Vercel 将“正式赞助 tailwindcss.com”；Google AI Studio 团队成员 Logan Kilpatrick 也表示其团队已成为 tailwindcss 项目的赞助方。

伊朗因经济恶化爆发的抗议演变为安全部队与抗议者的暴力事件，造成至少7人死亡。 死亡事件发生在洛雷斯坦省、恰哈尔马哈勒-巴赫蒂亚里省以及伊斯法罕省。官媒指责抗议者破坏政府大楼和银行。网传视频显示，街头聚集着抗议者并回荡着枪声。死者中有一名“动员穷人组织”志愿军成员。 总统佩泽希齐扬领导的政府虽然一直表明希望与抗议者谈判，但他也承认对货币贬值无能为力。伊朗塔斯尼姆通讯社日前报道说，伊朗政府经济团队正在召开紧急会议，讨论处理近期出现在汇率、贸易和民生方面的问题。 （美联社，新华社）

伊朗首都德黑兰1月8日晚爆发大规模抗议活动，人们高喊反对当局的口号。伊朗其他地区的示威及罢工罢市活动也仍在继续。抗议开始后，伊朗出现通信及网络中断的现象。
此前伊朗流亡王储礼萨·巴列维此前呼吁8日和9日晚举行示威。视频显示，至少部分抗议者响应了此次号召，呼吁巴列维回归。巴列维称，将根据人们对他呼吁的回应来提出进一步的计划。
有媒体称，伊朗安全部队将使用无人机来识别抗议参加者。但伊朗官员未对8日的抗议有任何回应。据报有安全官员受伤或死亡。
