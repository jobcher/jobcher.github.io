哪些因素影响日本人生育？

日本2023年总和生育率创出历史新低，哪些因素影响日本人生育？

《日本经济新闻》在5月以约5000名读者为对象进行了问卷调查，作为结婚和生育的阻碍因素，很多受访者提及“经济负担”（71.3%的）以及“兼顾工作的困难”(62.3%)。有40多岁的受访者表示：“在提高公司生产效率的名义下，夫妻双方的工作量均在增加，没有时间照顾孩子”。年富力强的这代人缺乏宽裕时间。

另一位40多岁的人士称：“本来就因为经济问题而无法结婚”，还有人对结婚本身感到犹豫。三菱UFJ研究咨询公司的首席研究员矢岛洋子表示，“由于近30年来实际工资下降以及子女抚养扣除制度的废除等原因，日本的育儿家庭的环境与欧美各国相比变得更加严峻”。

在调查中，近1成的人回答“无论采取什么措施都不想生育”。改变生育适龄世代的意识并非易事。

日本30年花了66万亿，仍未改变少子化

据日经中文网，日本少子化仍未得到遏制。6月5日，日本厚生劳动省公布的2023年总和生育率创出历史新低。2024年是政府首次采取少子化对策以来迎来的第30年，迄今为止投入的相关预算累计超过66万亿日元，但仍然看不到生育率止跌反弹。

日本政府制定首个综合对策“天使计划”是在1994年。契机是泡沫经济下的1989年生育率创出当时战后最低纪录的“1.57冲击”。当时出台了设立育儿休假津贴、扩充托儿所和实施延长保育等7项重点措施。

2003年实施了《少子化社会对策基本法》，2012年实施了《儿童及育儿支援法》等，2023年组建了儿童家庭厅。
