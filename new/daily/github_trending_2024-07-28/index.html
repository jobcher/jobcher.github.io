约翰霍普金斯大学政治学家洪源远（Ang Yuen Yuen）在对1978-2017年中国近5000条中央指令进行分析后，分别出三种指令：灰色指令（中央对允许或禁止某项政策措辞含糊不清）、黑色指令（中央明确鼓励某些政策）和红色指令（中央明确禁止某些政策）。

洪源远发现，这些指令出现了三种趋势：灰色指令持续增长，说明中央政策越来越含糊不清；黑色指令持续下降，表明中央明确鼓励政策指令越来越少；红色指令先降后升，说明明确禁令数量存在波动。

红色指令集中在公共安全（28%）、银行和证券（27%）以及土地管理（24%）方面，表明中央政府最有可能在涉及政治稳定、国家安全和公共资源的问题上划定红线。
