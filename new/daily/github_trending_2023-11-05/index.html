劳动收入份额是劳动者参与收入分配的结果反映。劳动收入份额受到劳动关系双方议价能力的支配，而议价能力又受到劳动市场与制度环境的影响。

补偿性工资理论认为，在完全竞争的劳动市场条件下，工作条件较为艰苦的劳动者将获得更高的劳动收入，以补偿恶劣工作环境带来的身心伤害。根据补偿性工资理论，较高的劳动收入份额是对超时劳动者的补偿，劳动收入份额与超时加班成正相关关系。
