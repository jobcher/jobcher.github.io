在中国大陆，麦当劳APP及小程序一度出现故障，麦当劳客服表示故障为全国范围的系统问题。当天下午4时许，麦当劳中国表示线上点餐系统已经全面恢复。
在香港，麦当劳发言人下午4时许回应称，公司电脑系统15日下午曾经发生故障，目前电脑系统已逐步回复正常。
在新加坡，App和门店的自助点餐机不可用，店员手写订单，亦有顾客支付现金。
在日本，受故障影响，门店收银系统无法使用，部分门店贴出了“临时休业”的告示，仍在营业的门店则通过顾客口头下单。麦当劳日本表示，正在紧急确认故障原因与范围，尚不清楚与其他国家和地区的故障原因是否相同。
（澎湃新闻，香港经济日报，NHK，中央社，卫报，亚洲新闻台）
