以色列议会1日以71票支持、10票反对的投票结果通过一项法律，允许政府以“危害国家安全”为名，临时关停在以色列运营的外国电视频道。以总理内塔尼亚胡表示将关停卡塔尔半岛电视台。
该法律赋予政府权力，可以临时禁止“被认为危害国家安全”的外国电视频道在以色列运营。根据这项法律，以通讯部长有权命令外国电视频道停止在以播放；关闭相关频道在以办事处并没收该频道设备；关闭该频道网站在以色列的服务器，或者以其他方式阻止对该网站的访问。关闭外国电视频道的最终决定需以色列安全内阁或政府批准。关停命令有效期为45天，但可以延长。
以色列总理内塔尼亚胡当天对这项立法表示欢迎，并称卡塔尔半岛电视台将“不再在以色列播放”。以色列通讯部长卡尔希说，“半岛电视台将在未来几天关闭”。卡尔希指责半岛电视台充当哈马斯的宣传武器，鼓励针对以色列的武装斗争。
半岛电视台随后发布声明称，这是以色列为压制半岛电视台发声所采取的系统性措施的一部分，所谓半岛电视台危害以色列安全的说法不实。半岛电视台重申，此类诽谤性指控不能阻止其继续勇敢且专业的报道，并将保留采取一切法律措施的权利。
美国国务院和白宫发言人均表态支持新闻自由和世界各地的独立自由媒体。虽然美国并不总是同意半岛电视台的报道，但尊重其工作。
半岛电视台是整个战争期间留在加沙的少数国际媒体之一，经常播放空袭和医院的血腥画面。半岛电视台称以色列屠杀。以色列则指责半岛电视台与哈马斯合作。
（半岛电视台，新华社，美联社）

伊朗驻叙利亚大使馆4月1日遭以色列导弹袭击，袭击造成至少5人死亡。
叙利亚军方说，当天17时许，以色列从其占领的戈兰高地方向对伊朗领事部门发动导弹袭击。伊朗驻叙利亚大使馆新闻处说，袭击发生时，包括多名伊朗军事顾问在内的受邀人员正在楼里准备参加开斋宴。
伊朗大使侯赛因·阿克巴里表示，导弹击中了使馆内的一座领事馆大楼，他的住所位于最上面两层。现场可见，使馆内一栋被毁建筑的废墟上，救援人员正在现场清理废墟。
根据阿克巴里和伊斯兰革命卫队的说法，5至7人在袭击中死亡，其中包括革命卫队下属“圣城旅”的高级指挥官穆罕默德-礼萨·扎赫迪及其副手哈吉·拉黑米等人。真主党发言人称，真主党成员侯赛因·优素福也在袭击中死亡。
伊朗认为扎赫迪是此次的袭击目标。扎赫迪是叙利亚的一名军事顾问，2016年之前一直担任黎巴嫩和叙利亚“圣城旅”的负责人。
阿克巴里对以色列的袭击行为表示谴责。他说，伊朗对此次袭击“不会没有回应”。叙利亚外长梅克达德对这起空袭事件予以“最强烈措辞”的谴责。伊朗外交部强调，伊朗保留对袭击采取反制措施的权利。
虽然以色列军方对此不予置评，但《纽约时报》报道称，多名以色列官员承认了此次袭击。以色列还指示其驻世界各国的大使馆加强安全措施。
（路透社，美联社，新华社）

2月底，湖南一女规培生在该省某医院刎颈身亡，距离规培结业仅余数月；3月，广西又有一名男规培生在医院以类似方式离世。规培生处境引发大众关注。
多名受访者认为，付出多、获得少，导致部分规培生特别是专硕生群体消极情绪较为突出。一些规培生会抱怨从事基础性事务。专硕生由于仍是在校学生身份，会面临“谈工作时是医生，讲待遇时是学生”的“双标”境况。
同时，学医的沉没成本高、退路选择窄。多家规培医院的招生简章明确，如果“退培”，3年内不可再参加规培。有的地方还要求离院前退还相关各项待遇、福利。一名专硕生表示，如果中途放弃，可能只剩考公和去医药公司两种选择。
多位医学界人士恳言，从医这条路并不容易，压力大、责任重是常态，建议适当改善规培生尤其是专硕生待遇，多措并举构建正向包容的规培环境。
（新华每日电讯）
