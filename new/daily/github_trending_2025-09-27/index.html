LoopDNS资讯播报:

特朗普签署价值140亿美元的TikTok美国交易

美国总统唐纳德·特朗普周四签署了一项行政命令，批准了一项使TikTok在美国继续运营的交易，前提是中国方面同意。该协议将成立一家新的合资企业来监管TikTok的美国业务，字节跳动持股不足20%。甲骨文（Oracle）、银湖资本（Silver Lake）和MGX基金合计控制约45%，而字节跳动的投资者则保留35%。甲骨文将负责安全运营和云服务，CEO拉里·埃里森（Larry Ellison）将直接参与。副总统JD·万斯（JD Vance）表示，该业务估值为140亿美元。特朗普称，这项交易确保了TikTok“完全由美国人运营”。

字节跳动的投资者，包括General Atlantic、Susquehanna和红杉资本（Sequoia），将向新的美国实体注入股权。该安排满足了美国国家安全法对字节跳动在美资产剥离的要求，美国将控制推荐算法，且不会有联邦股权参与。特朗普还提到，鲁珀特和拉克兰·默多克，以及戴尔公司CEO迈克尔·戴尔可能参与其中。

来源：彭博社
