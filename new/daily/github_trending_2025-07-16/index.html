GitHub Copilot在VS Code中正式支持模型上下文协议(MCP)

GitHub Copilot在VS Code中正式支持模型上下文协议(MCP)

官方消息显示，从 VS Code 1.102（2025 年 6 月版本） 起，GitHub Copilot 对 Model Context Protocol (MCP) 的支持已从预览转为正式可用，开发者可在生产环境中将 MCP 服务器接入工作流，调用数据库、API 等外部资源，为 Copilot 提供更丰富的上下文。

MCP 现在被视为 VS Code 配置文件中的一等资源，可通过全新的 MCP 视图与 Gallery 进行管理与安装，并与 Settings Sync 同步。微软称，MCP 的通用接口让授权、提示、采样和资源管理统一规范，方便构建 AI 代理型集成。

同时，GitHub 已将 Copilot Chat 扩展开源，允许社区贡献并创建自定义聊天模式和工具链；其他编辑器（Visual Studio、Eclipse、JetBrains IDE 等）的 MCP 支持仍处公测阶段。

特朗普政府在上诉时辩称，政府承认只有国会才能取消教育部，而相关裁员只是为了“精简”该部门，这属于行政部门而非地区法院的职权范围。原告则表示，如果允许政府拆解教育部，即使后续原告获胜，也无法消除拆解所造成的大部分伤害。
最高法院14日以6比3的结果否决了马萨诸塞州联邦地区法院法官的禁令。六名保守派大法官并未给出否决原因。三名自由派大法官持反对意见，其中，大法官索尼娅·索托马约尔称：“当行政部门公开宣布其打算违法并付诸实施时，司法部门的责任是制止、而不是加速这种非法行为。”
此外，由加利福尼亚州牵头的20多个州政府14日起诉特朗普政府，要求解冻约68亿美元教育经费。联邦政府应在7月1日前向各州拨付这笔资金，但教育部6月30日通知各州，无法按期拨款且未提供合理解释。白宫官员先前表示，这些教育拨款正接受审查，初步调查发现，一些教育拨款被用于“激进的左翼议程”。
（美联社，NPR，新华社 1，2）
