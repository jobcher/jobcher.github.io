@Easonlee的AI笔记:
要点：
• 当前AI模型评估表现与实际经济影响存在显著脱节，模型“看起来聪明”但经济效益滞后。
• 强化学习（RL）训练可能因过于聚焦评估指标而导致模型泛化能力不足，如同“竞技编程学生”过度专业化。
• 人类学习在样本效率和泛化能力上远超AI，这可能归因于进化赋予的先验知识和内建的“价值函数”（情感）。
• AI发展正从追求“扩展”的时代回归“研究”时代，需要探索新的训练方案和更高效的计算资源利用方式。
• 通用人工智能（AGI）并非一蹴而就的成品，而是一个持续学习、不断进化的过程，类似“渴望学习的15岁孩子”。
• AI的强大力量难以被完全想象，因此逐步部署和实际展示其能力，对于公众和政府理解并适应至关重要。
• AI安全问题应通过实际部署、发现并纠正故障来增强系统鲁棒性，而非仅依赖理论上的安全思考。
• 构建能“关爱有感知生命”的AI，可能比仅关爱人类更容易，且需思考如何有效约束超级智能的力量。
• AI研究的“品味”在于追求美、简洁、优雅，并从大脑中汲取正确启发，形成支撑研究方向的“自上而下”信念。
• 自我对弈在数据稀缺时有潜力，尤其适用于发展特定技能（如谈判、策略），可作为通用AI训练的补充。

阿里通义实验室开源高效图像生成模型 Z-Image-Turbo

阿里巴巴通义实验室宣布开源图像生成基础模型 Z-Image-Turbo，并在 Hugging Face 等平台上线。该模型属于 Z-Image（造相）系列，参数规模约 60 亿，采用单流 Diffusion Transformer 架构，主打高效推理与通用图像生成能力。

官方介绍称，Z-Image-Turbo 通过 8 步推理即可完成图像生成，在企业级 H800 GPU 上可实现亚秒级延迟，并可部署于 16GB 显存消费级设备。模型主要面向写实图像生成、中英文双语文字渲染以及复杂指令对齐场景，支持通过 Hugging Face diffusers 直接加载使用，许可证为 Apache-2.0，可用于商业应用与二次开发。

香港大埔宏福苑多栋高层建筑发生火灾，至少13人死亡，多人被困，火势仍未受控


香港新界大埔王福苑于26日下午约2时50分发生严重火灾，火势从维修外墙的脚手架迅速蔓延，波及屋苑内多栋高层住宅楼。王福苑共有8栋大楼、近2000个单位，居住约4000名居民，其中长者比例高，约36%居民为65岁或以上，受影响人数达数千。

截至目前，火警仍未完全扑灭，已有至少13人死亡，包括一名殉职消防员，另有多人受伤及失联。由于火势强劲、烟雾浓厚以及外墙棚架坍塌风险，消防员仍无法进入部分高层单位，但已与部分被困居民取得联系，并指导他们以胶带与湿纸巾封堵门窗等待救援。

所有撤离居民今晚均无法返回家园，需前往临时安置点或另觅住所。消防与救援行动仍在持续。
