韩国低生育率背后的真相——教育的地位外部性与低收入群体生育率

在大多数国家和不同时期，生育率与收入通常呈负相关，无子女率也常常与家庭收入呈正相关关系。然而，在韩国，最低收入五分位组的平均子女数量为1.80，而最高收入五分位组则为2.03。这一发现表明，韩国的低生育率与家庭收入密切相关。

我们以私立教育溢出效应的新经验证据为契机，提出了一种将这些事实与地位外部性和内生生育率联系起来的新理论。我们发现，如果消除地位外部性，生育率会提升28%，低收入群体的生育率增幅会更高。
