一线城市毕业季租金全部转涨

据一财，毕业季租房需求的释放，让全国大中城市的租金呈现了企稳回升的态势。日前，诸葛找房数据研究中心发布的数据显示，今年6月，全国40个大中城市租金挂牌均价为34.81元/平方米/月，环比上升0.13%，时隔4个月，环比数据再度转正。租金企稳回升的原因在于，6月进入毕业季，租赁需求释放增多。诸葛找房数据研究中心预计，由于7月份仍处于毕业季，7月租金或延续稳中有升态势。尽管租金环比已经实现回升，但与历史租金水平对比，全国40个大中城市当前租金水平整体并不及前两年，同比下跌1.82%，跌幅较上月收窄0.14个百分点。其中，一、二线城市租金同比已分别连续下滑12个月和8个月。诸葛找房数据研究中心认为，这主要受住房消费降级以及租赁市场供求关系改变等因素影响，租金缺乏上升动力。

巴黎奥运会日本选手部分参赛服能防红外线偷拍

作为巴黎奥运会14个比赛项目的日本代表队的参赛服和训练服，美津浓将提供可防止红外线相机偷拍内衣等的服装。该服装使用特殊纤维吸收人体散发的红外线，即使被红外相机拍摄，也难以透视内衣或身体。这有助于防止运动员被偷拍的问题，协助创造运动员可以集中精力投身比赛的环境。该服装采用了含有吸收红外线的特殊矿物的纤维“Dry Aero Flow Rapid”。吸收的红外线有助于促进汗液蒸发，不仅可以防偷拍，还具有良好的散热性能。预计日本代表队的运动员将在巴黎奥运会女子排球及女子乒乓球等6个比赛项目中穿着这款比赛服。
