瑞幸咖啡传进军香港 据报3万港元请店长

社交平台小红书疯传，有“中国星巴克”之称的瑞幸咖啡(Luckin Coffee)有意进军香港市场，将于油尖旺及中环地区开设分店，并称其招聘海报列出将聘请包括店长或副店长等职级，据报月薪最高达3万港元，咖啡师最高亦有2万港元。
资料显示，瑞幸咖啡创立于2017年，为全国最大咖啡连锁品牌之一。由神州优车创始人陆正耀和COO钱治亚共同创立，总部位于厦门。截至今年7月底，全国门店数目已突破2万间。该品牌曾推出多款爆红饮品，包括2020年的“厚乳拿铁”、2021年的“生椰拿铁”等，去年与贵州茅台联乘的“酱香拿铁”更吸引不少人抢购，开售首日售出逾542万杯。
除瑞幸咖啡外，内地不少饮品店近日持续来港发展，包括霸王茶姬于K11购物艺术馆开店。据悉，顾客需轮候2小时才取得所买茶饮，茶百道早前已来港拓展业务，在旺角落户；另外，内地茶饮店“茶二”亦选址K11 MUSEA开业。
