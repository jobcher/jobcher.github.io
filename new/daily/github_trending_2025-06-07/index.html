乌克兰军方称，6日清晨对俄罗斯萨拉托夫州的恩格斯空军基地和梁赞州的佳吉列沃军用机场进行了打击，正在核实俄军的损失情况。乌军还对萨拉托夫州的3座燃油和润滑油油库进行多轮打击，相关设施随后起火并连续发生爆炸。此外，乌军与乌国防部情报总局配合，对俄境内和乌被占领土上的多处俄军事设施进行了打击。
