QEMU 11.0 正式发布：增强虚拟化安全与硬件扩展支持

QEMU 11.0 正式发布：增强虚拟化安全与硬件扩展支持

开源虚拟化工具 QEMU 11.0 版本现已发布。新版本显著提升了安全性与兼容性，包括在 KVM 模式下新增对英特尔控制流强制技术（CET）虚拟化的支持，并支持 AMD SEV-SNP 与 Intel TDX 虚拟机的重置功能。此外，新版引入了原生 Nitro Enclaves 加速器以支持 AWS 环境，并增加了英特尔至强 Diamond Rapids 及 MIPS P8700 等处理器模型。

在架构优化方面，新版本更新了 RISC-V 与 LoongArch 的指令集扩展支持，同时清理了过时的 i440FX 和 Q35 机器模型。此次更新进一步强化了 QEMU 在 Linux 虚拟化技术栈中的核心地位，为跨平台模拟与云端安全提供了更完善的硬件仿真支持。

来源：phoronix / QEMU

高盛研报解析DeepSeek V4对中国AI产业影响
