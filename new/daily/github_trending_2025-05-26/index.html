朝鲜驱逐舰下水事故 已知3责任人被拘留

据官媒朝中社周日报导称，驱逐舰下水事故的调查工作持续深入开展。司法机关拘留了对事故负有责任的清津造船厂总工程师姜正哲、船体总装车间主任韩京学和行政副厂长金勇鹤。

报导称，事故调查组周六（24日）向朝鲜劳动党中央军事委员会汇报了工作情况。目前没有新发现的舰船损坏，现场修复推进小组正在按计划进行修复作业。

朝中社此前表示，司法机关将先拘留、调查对事故有明显责任的人员。清津造船厂厂长洪吉浩已于周四（22日）被司法机关传唤。

朝中社报导称，事故舰体受损情况没有之前公布的那么严重。船底并未出现破孔，船体右舷被刮蹭，通过船尾部分的救生通道流进一定容量的海水。朝鲜专家认为，抽出浸水隔室的海水，从船台脱离船首部分来改善舰船稳性需要两三天时间，修复舷侧需要十余天。

朝鲜最高领导人金正恩日前在清津造船厂观摩朝军5000吨级驱逐舰下水仪式，但驱逐舰下水时发生严重事故，造成舰体受损。据悉，事故军舰或与朝鲜上月25日下水的5000吨级多功能攻击型驱逐舰“崔贤”号属于同一级别。

金正恩目睹了事故全过程，并对此进行了严厉批评，称这起事故完全是由于疏忽和不负责任，不尊重科学的经验主义而产生的无法容忍的重大事故和犯罪行为，要求务必在6月朝鲜劳动党中央委员会全会召开前完成修复工作。
