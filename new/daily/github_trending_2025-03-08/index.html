浙大学者研究揭示中国地方政府GDP数据造假：年均虚增0.24个百分点

浙江大学公共管理学院龚斌磊、沈宇辉与陈帅教授团队在《Journal of Public Economics》（《公共经济学杂志》）发表研究，通过创新性“比率形式”与“扎堆估计法”量化了中国地方政府GDP数据造假现象。分析显示，2000-2018年间，约9.59%的地级市GDP增速存在人为调整以达到预设目标，导致年均增速虚高0.24个百分点，累计至2018年全国GDP总量被高估约4.1%。研究指出，2013年后中央降低经济绩效考核权重、推广弹性目标设定（如“增长X%左右”）等政策显著遏制了造假行为，窗口期内操纵概率从42.12%骤降至12.61%。学者建议进一步弱化GDP考核刚性，转向多元发展指标，以根治“数字政绩”痼疾。

因涉嫌带头发动内乱而被逮捕起诉的韩国总统尹锡悦3月7日获释。
首尔中央地方法院刑事合议25庭当天批准尹锡悦的撤销逮捕措施申请，尹锡悦将在非羁押状态下继续受审。这距离尹锡悦被羁押时过51天，也距离其被逮捕起诉时过40天。
法院认为，羁押期限应以小时而非天数为单位计算，尹锡悦拘留合法性审查的相关材料等在法院的停留时间也被计入羁押时间，法院由此认为，尹锡悦在羁押期限届满后被检方起诉。
法院还提及尹锡悦辩护人的主张，即公调处侦查范围不含内乱罪，且公调处和检察厅是两个独立的侦查机构。然而，双方在没有法律依据的情况下相互协商分期适用《刑事诉讼法》所规定的羁押期，在此过程中未走所需程序。法院表示，有必要进一步明确相关流程并消除围绕侦办过程合法性的疑点，故决定撤销逮捕。
（韩联社）

欧洲理事会3月6日同意欧盟委员会主席冯德莱恩提出的“重新武装欧洲”计划。通过放宽预算限制，使有意愿的欧盟国家可以增加国防开支；敦促欧盟委员会提升所有国家国防开支；提供贷款用于成员国的国防开支。
公报说，俄乌冲突对欧盟构成了生存挑战，欧洲必须更加负责地处理自身防务问题。欧盟将加强其整体国防能力，减少其战略依赖，解决其关键能力差距，并相应地加强整个联盟的国防技术和工业基础，以便能够更好地供应所需的装备。
法国总统马克龙3月5日提议将法国核武库提供的保护扩大到欧洲伙伴。对于该提议，欧洲领导人虽然表现了开放的态度，但也颇为谨慎。
乌克兰总统泽连斯基将于下周前往沙特阿拉伯会见王储，他的团队将留下与美国官员会谈。
