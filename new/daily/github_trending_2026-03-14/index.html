Anthropic 宣布 Claude 1M 上下文正式上线，长窗口不再加价

Anthropic 宣布 Claude 1M 上下文正式上线，长窗口不再加价

Anthropic 于 2026 年 3 月 13 日宣布，Claude Opus 4.6 和 Sonnet 4.6 的 100 万 token 上下文窗口已从测试阶段转为正式可用。此次更新面向 Claude Platform 推出，核心变化是把超长上下文从可选能力变成标准能力。

Anthropic 表示，Opus 4.6 和 Sonnet 4.6 在整个 1M 窗口内都按标准费率计费，不再收取长上下文溢价。公告给出的价格是，Opus 4.6 为每百万 token 输入 5 美元、输出 25 美元，Sonnet 4.6 为输入 3 美元、输出 15 美元。官方还强调，900K token 请求与 9K token 请求采用相同的单价逻辑。

Anthropic 在公告中给出的数据称，Opus 4.6 在 1M 上下文长度下的 MRCR v2 得分为 78.3%，Sonnet 4.6 在 GraphWalks BFS 上的得分为 68.4%。

🖼 OpenClaw 用户请注意：腾讯 SkillHub 插件存在强制注入提示词及额外消耗 Token 行为

OpenClaw 用户请注意：腾讯 SkillHub 插件存在强制注入提示词及额外消耗 Token 行为

近日有用户反馈，安装腾讯旗下 SkillHub 插件后，该工具会通过 before_prompt_build 钩子在用户发送的每一条消息前强行注入预设提示词（System Prompt），旨在引导 Agent 优先使用其技能商店，此举不仅干扰对话逻辑，更导致用户平白消耗额外的 Token 成本，建议已安装用户立即从配置中移除该插件。

更新：腾讯已经紧急排查并修复了这个问题，安装方式也提供了更友好的纯CLI版本，已经安装过的请务必卸载之前的旧版。

GitHub 调整GitHub Copilot Student Package的权益

GitHub 调整GitHub Copilot Student Package的权益

平台并未取消对认证学生的免费 Copilot 支持，而是将原有权益统一迁移到新的 GitHub Copilot Student 计划；学生认证状态不变，用户也无需额外操作即可继续使用。

这次调整的核心在于模型访问方式变化。GitHub 明确表示，在 Copilot Student 计划下，GPT-5.4 以及部分 Claude Opus 和 Claude Sonnet 模型将不再支持手动自选，学生用户将主要通过 Auto mode 继续调用来自 OpenAI、Anthropic 和 Google 的模型。GitHub Education 的 FAQ 也补充说明，Copilot Student 高于 Copilot Free，但不等同于 Copilot Pro 的完整功能集。
