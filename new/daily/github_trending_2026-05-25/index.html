з«№ж–°зӨҫ:

дјҠжң—5жңҲ10ж—ҘйҖҡиҝҮе·ҙеҹәж–ҜеқҰжӢ’з»қзҫҺеӣҪжҸҗеҮәзҡ„ж–№жЎҲпјҢи®ӨдёәеҗҢж„ҸиҜҘж–№жЎҲе°Ҷж„Ҹе‘ізқҖдјҠжң—еұҲд»ҺдәҺзү№жң—жҷ®зҡ„иҝҮеҲҶиҰҒжұӮгҖӮ ж №жҚ®дјҠжң—еӘ’дҪ“жҠҘйҒ“пјҢдјҠжң—иҰҒжұӮзҫҺеӣҪж”Ҝд»ҳжҲҳдәүиө”еҒҝпјҢе№¶зЎ®и®ӨдјҠжң—еҜ№йңҚе°”жңЁе…№жө·еіЎзҡ„дё»жқғгҖӮдјҠж–№иҝҳејәи°ғпјҢжңүеҝ…иҰҒз»“жқҹеҜ№дјҠжң—зҡ„еҲ¶иЈҒпјҢе№¶и§ЈеҶ»иҜҘеӣҪиў«еҶ»з»“зҡ„иө„дә§гҖӮ зү№жң—жҷ®е№¶жңӘйҖҸйңІдјҠжң—еӣһеә”зҡ„з»ҶиҠӮпјҢдҪҶд»–еҜ№дјҠжң—зҡ„еӣһеә”иЎЁиҫҫдёҚж»ЎпјҢжҢҮиҙЈдјҠжң—еңЁвҖңжҲҸеј„вҖқзҫҺеӣҪпјҢз§°е…¶вҖңе®Ңе…ЁдёҚеҸҜжҺҘеҸ—вҖқгҖӮ дёҖиүҳиҙ§иҲ№еңЁеҚЎеЎ”е°”йҷ„иҝ‘жө·еҹҹиЎҢй©¶ж—¶пјҢиў«з–‘дјјж— дәәжңәеҮ»дёӯиө·зҒ«гҖӮйҳҝиҒ”й…Ӣе’Ңз§‘еЁҒзү№жҠҘе‘Ҡжңүж— дәәжңәиҝӣе…ҘйўҶз©әгҖӮйҳҝиҒ”й…ӢеҮ»иҗҪ2жһ¶ж— дәәжңәгҖӮ жҚ®дјҠжң—еӘ’дҪ“жҠҘйҒ“пјҢдјҠжң—жңҖй«ҳйўҶиў–з©Ҷжқ°еЎ”е·ҙдјҡи§ҒдјҠжң—жӯҰиЈ…йғЁйҳҹе“ҲеЎ”вҖҰ

зҫҺеӣҪй«ҳзә§е®ҳе‘ҳ5жңҲ24ж—Ҙз§°пјҢдјҠжң—вҖңеҺҹеҲҷдёҠвҖқеҗҢж„ҸејҖж”ҫйңҚе°”жңЁе…№жө·еіЎгҖҒж”ҫејғжө“зј©й“Җеә“еӯҳпјҢд»ҘжҚўеҸ–зҫҺеӣҪи§ЈйҷӨжө·дёҠе°Ғй”ҒгҖӮдјҠжң—жңҖй«ҳйўҶиў–з©Ҷжқ°еЎ”е·ҙе·Іи®ӨеҸҜеҚҸи®®зҡ„еӨ§иҮҙжЎҶжһ¶гҖӮ

зү№жң—жҷ®иЎЁзӨәпјҢи°ҲеҲӨвҖңиҝӣеұ•жңүеәҸдё”д»Ҙе»әи®ҫжҖ§зҡ„ж–№ејҸвҖқиҝӣиЎҢпјҢдёҺдјҠжң—зҡ„е…ізі»еҸҳеҫ—вҖңжӣҙеҠ дё“дёҡе’ҢеҜҢжңүжҲҗж•ҲвҖқгҖӮдҪҶд»–е‘ҠиҜүи°ҲеҲӨд»ЈиЎЁпјҢдёҚжҖҘдәҺдёҺдјҠжң—иҫҫжҲҗд»»дҪ•еҚҸи®®гҖӮ

дјҠжң—иҷҪз„¶е°ҡжңӘе…¬ејҖзЎ®и®ӨеҚҸи®®пјҢдҪҶдјҠжң—жҖ»з»ҹдҪ©жіҪеёҢйҪҗжү¬иЎЁзӨәпјҢдјҠжң—еҮҶеӨҮвҖңеҗ‘дё–з•ҢдҝқиҜҒпјҢжҲ‘д»¬дёҚжҳҜеңЁиҝҪжұӮж ёжӯҰеҷЁвҖқгҖӮдјҠжң—еӨ–дәӨйғЁеҸ‘иЁҖдәә24ж—ҘиЎЁзӨәпјҢзҫҺдјҠз«Ӣеңәзҡ„еҲҶжӯ§жӯЈеңЁзј©е°ҸгҖӮ
пјҲзҫҺиҒ”зӨҫпјҢи·ҜйҖҸзӨҫпјү

вҶ©пёҸ еӨҡеҗҚж—Ҙжң¬ж”ҝеәңж¶ҲжҒҜдәәеЈ«йҖҸйңІпјҢд№ иҝ‘е№іеңЁ14ж—ҘдёҺзү№жң—жҷ®зҡ„дјҡи°ҲдёӯпјҢзӮ№еҗҚй«ҳеёӮж—©иӢ—е’Ңиө–жё…еҫ·дәҢдәәеЁҒиғҒең°еҢәе’Ңе№іпјҢжҢҮиҙЈж—Ҙжң¬еҶҚеҶӣдәӢеҢ–дёҘйҮҚеЁҒиғҒең°еҢәе’Ңе№ізЁіе®ҡпјҢе№¶еңЁжӯӨеҹәзЎҖдёҠ…

з«№ж–°зӨҫ:

зҫҺеӣҪжө·еҶӣйғЁд»ЈзҗҶйғЁй•ҝй«ҳйӣ„иЎЁзӨәпјҢзҫҺеӣҪжӯЈеңЁжҡӮеҒңеҜ№еҸ°ж№ҫд»·еҖј140дәҝзҫҺе…ғзҡ„еҶӣе”®пјҢд»ҘзЎ®дҝқзҫҺеҶӣжӢҘжңүи¶іеӨҹеј№иҚҜз”ЁдәҺдјҠжң—жҲҳдәүгҖӮ еңЁеҸ°еҢ—пјҢжҖ»з»ҹеәңеҸ‘иЁҖдәәз§°е°ҡжңӘж”¶еҲ°зӣёе…ідҝЎжҒҜгҖӮеңЁеҢ—дә¬пјҢеӨ–дәӨйғЁеҸ‘иЁҖдәәз§°пјҢеқҡеҶіеҸҚеҜ№зҫҺеӣҪеҗ‘еҸ°ж№ҫеҮәе”®жӯҰеҷЁзҡ„з«ӢеңәдёҖиҙҜгҖҒжҳҺзЎ®гҖҒеқҡе®ҡгҖӮ пјҲBBCпјҢдёӯеӣҪеӨ–дәӨйғЁпјү

еӨҡеҗҚж—Ҙжң¬ж”ҝеәңж¶ҲжҒҜдәәеЈ«йҖҸйңІпјҢд№ иҝ‘е№іеңЁ14ж—ҘдёҺзү№жң—жҷ®зҡ„дјҡи°ҲдёӯпјҢзӮ№еҗҚй«ҳеёӮж—©иӢ—е’Ңиө–жё…еҫ·дәҢдәәеЁҒиғҒең°еҢәе’Ңе№іпјҢжҢҮиҙЈж—Ҙжң¬еҶҚеҶӣдәӢеҢ–дёҘйҮҚеЁҒиғҒең°еҢәе’Ңе№ізЁіе®ҡпјҢе№¶еңЁжӯӨеҹәзЎҖдёҠж•Ұдҝғзү№жң—жҷ®дёҚиҰҒж”ҜжҢҒдәҢдәәгҖӮ
зү№жң—жҷ®еҜ№жӯӨеҲҷиЎЁзӨәпјҢй«ҳеёӮвҖңжҳҜдҪҚеҫҲжЈ’зҡ„йўҶеҜјдәәвҖқгҖӮ
ж—Ҙжң¬ж”ҝеәңдёҖдҪҚй«ҳе®ҳиҜ„д»·йҒ“пјҡвҖңзү№жң—жҷ®еңЁд№ иҝ‘е№ійқўеүҚеҸ‘иЎЁдәҶжөҒйңІеҮәеҜ№йҰ–зӣёдҝЎд»»зҡ„иЁҖи®әпјҢеҗ‘дёӯеӣҪеұ•зӨәдәҶж—ҘзҫҺдёӨеӣҪзҡ„еӣўз»“гҖӮвҖқ
зү№жң—жҷ®з»“жқҹи®ҝеҚҺеҗҺзҡ„15ж—ҘпјҢеңЁз©әеҶӣдёҖеҸ·дёҠеҗҢй«ҳеёӮж—©иӢ—з”өиҜқдјҡи°ҲгҖӮдјҡи°Ҳз»“жқҹеҗҺпјҢй«ҳеёӮеҜ№и®°иҖ…иЎЁзӨәпјҡвҖңпјҲзү№жң—жҷ®пјүз»ҷдәҲдәҶж—Ҙжң¬йқһеёёеӨ§зҡ„её®еҠ©пјҢжҲ‘еҜ№жӯӨж·ұиЎЁж„ҹи°ўгҖӮвҖқдҪҶе…¶е№¶жңӘйҖҸйңІдјҡи°Ҳзҡ„е…·дҪ“еҶ…е®№гҖӮ
пјҲиҜ»еҚ–ж–°й—»пјҢж—Ҙжң¬з”өи§ҶеҸ°пјү

еҠ е·ҙеҫ·еңЁзҫҺж”ҝеқӣй•ҝжңҹд»ҘдәІжҷ®дә¬гҖҒдәІйҳҝиҗЁеҫ·пјҢжҢҒеӯӨз«Ӣдё»д№үи§ӮзӮ№иҖҢй—»еҗҚгҖӮеҘ№еңЁ2020е№ҙжҖ»з»ҹеҲқйҖүдёӯжӣҫеҜ»жұӮж°‘дё»е…ҡжҸҗеҗҚпјҢеӨұиҙҘеҗҺиҪ¬жҠ•е…ұе’Ңе…ҡпјҢеңЁ2024е№ҙиғҢд№Ұзү№жң—жҷ®гҖӮдҪҶжҳҜеҘ№дёҠд»»еҗҺеӨҡж¬ЎдёҺзү№жң—жҷ®зҡ„еҜ№дјҠејәзЎ¬и·ҜзәҝиғҢзҰ»пјҢе…¶еңЁж—©еүҚеңЁеӣҪдјҡз§°дјҠжң—е№¶жңӘжӢҘж ёзҡ„иҜҒиҜҚиў«зү№жң—жҷ®е…¬ејҖй©іж–ҘпјҢд»ҺеҺ»е№ҙиө·е·Із»Ҹиў«й•ҝжңҹиҫ№зјҳеҢ–пјҢж¶үдјҠжң—гҖҒе§”еҶ…з‘һжӢүгҖҒеҸӨе·ҙзӯүеҶізӯ–еқҮи·іиҝҮеҘ№е®һж–ҪгҖӮ
