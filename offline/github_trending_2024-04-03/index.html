韩国政府推迟吊销未返岗实习住院医师的行医执照；医学院教授陆续提交辞呈。 韩国政府原计划26日对未返岗的实习住院医师给予吊销行医执照的处分。但总统办公室相关人士25日表示，政府正与医界开展对话，以谋求灵活的解决方案，故目前不宜吊销相关医生的行医执照，此举并不意味着处分时间将无期限推迟。 韩国医学院教授们25日起陆续提交辞呈，已有部分医学院内近百名教授集体请辞，还有部分医学院教授召开大会，纷纷辞职。对于前一天教授组织与执政党之间的座谈会，各大医学院教授谴责毫无实际内容。 （韩联社 1，2）

韩国总统尹锡悦2日提出，愿亲自与反对医学生扩招的实习住院医师对话讨论医改。
尹锡悦在1日就医改发表对国民谈话时表示，若拿出妥当合理的方案的话，可与医疗界讨论，也可就缩小扩招规模进行讨论。尹锡悦还提议构建社会协商机制，以便医疗界、利害关系方、政府就扩招等医改方案进行协调。总统室据此解释称，不执着于扩招2000名医学生这一数字，将同医疗界一道寻求合理方案。总统室相关人士表示，总统真心愿意对话，多次强调为医改需要与医疗界积极对话。
首尔行政法院2日驳回了全国医科大学教授协议会提出的停止执行扩招申请。法院表示，医学院教授在法律上与扩招无直接利益关系，不具备申请人资格。
法院还表示，现行《高等教育法》中不存在关于决定招生规模时顾及教授利益的相关规定。即使申请人在提供优质教育时因医学院扩招受阻，这也是由各所高校的教育条件所致，应通过各高校扩充设施和教员等方式加以解决。
（韩联社 1，2）

拜登、习近平通电话，共同应对世界局势紧张和机遇，寻求中国在乌克兰战争和中东动荡中的合作

美国总统乔·拜登和中国领导人习近平周二通过电话交谈，这是自去年11月在加州会面以来他们首次进行一对一的交流。

尽管两国之间存在深刻的彼此猜忌，以及两国政府正在实施出口管制、制裁和关税，但这两个全球最大经济体的领导人成功地保持了外交稳定的外表。与此同时，他们还在人工智能带来的风险以及打击芬太尼等非法毒品问题上寻求共同立场。

一位向记者通报情况的高级行政官员表示，拜登预计将敦促习近平加强禁毒努力，同时讨论自去年11月以来在限制和干扰一些合成毒品前体化学品流动方面所取得的进展。

该官员还表示，拜登还将与习近平重申，中国应利用其与俄罗斯和伊朗的关系，促进乌克兰战争周边以及中东动荡地区的稳定。他补充说，总统还将就中国在南中国海的行为提出关切，包括最近一次中国对菲律宾供应船进行水炮轰击的事件。

两位领导人在去年11月同意保持更加频繁的交流，因为他们此前一对一会谈已有一年时间，期间包括中国气球飞越美国的事件。拜登将面临对阵唐纳德·特朗普的连任竞选，特朗普在首次上任后发动了对中国的贸易战，并提议如果他再次上台，将对所有中国进口商品征收60%的关税。
