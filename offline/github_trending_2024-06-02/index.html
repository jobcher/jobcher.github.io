美国总统拜登5月31日介绍了以色列提出的三阶段停火提议，得到哈马斯的积极回应。
第一阶段是为期6周的“完全停火”，其间以军从加沙所有人口聚集区撤出；哈马斯释放妇女、老人和伤员等部分被扣押人员；以色列释放数百名被关押的巴勒斯坦人。
以色列和哈马斯就实现“永久停止敌对行动”的必要安排进行谈判。如果谈判未能在6周内取得成果，停火期限将延长，美国、卡塔尔和埃及也将确保谈判能够持续进行。
拜登表示，只要哈马斯遵守承诺，“暂时停火”就会变成“永久停止敌对行动”。从第一阶段进入第二阶段，还有很多“细节需要谈判”。
第二阶段哈马斯将释放剩余被扣押人员。作为交换条件，以军将全部撤出加沙地带。永久停火将开始。第三阶段将开启加沙地带的大规模重建计划，并将人质的遗骸归还其家属。
哈马斯称“积极看待”相关提议。它准备以“积极和建设性的方式”参与任何基于永久停火、以色列撤军、加沙重建、难民返乡以及“真正”换囚协议的提议，但前提是以色列明确对协议的承诺。
以色列总理办公室表示，总理内塔尼亚胡已授权其谈判团队提出该提议，但坚称在释放所有被扣押人员、消灭哈马斯军事和管理能力等目标实现之前，战争不会结束。
（路透社，新华社）
