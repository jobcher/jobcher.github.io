OpenAI 发布 GPT-5.4 与 GPT-5.4 Pro，覆盖 ChatGPT、API 和 Codex

OpenAI 将 GPT-5.4 定位为面向专业工作的前沿模型，称其整合了推理、编码与智能体工作流的近期进展，并吸收 GPT-5.3-Codex 的编码能力以提升跨工具与软件环境的工作表现。OpenAI 同时强调其推理阶段的 token 效率，相比 GPT-5.2 用更少 token 完成问题求解，以降低用量并提升速度。

ChatGPT 端新增可控性机制：面对更长、更复杂的要求，GPT-5.4 Thinking 会先给出工作前导说明，用户可以在生成过程中插入指令或调整方向，以减少“推倒重来”或追加多轮对话的需求。该能力已在 chatgpt.com 与 Android 上线，iOS 版本标注为即将推出；OpenAI 也称模型在更长时间思考时能更好保持对早先上下文的把握。
