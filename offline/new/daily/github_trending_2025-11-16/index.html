前财政部长谈房地产税：立法程序已完成，关键问题已解决

在今日举行的第16届财新峰会上，中国前财政部部长楼继伟谈及房地产税问题。他表示，房地产税是最适合作为地方税种的税，因为一个地区公共服务越好，对应的房产税也越高，两者形成相互促进的关系。

楼继伟透露，房地产税的概念最早于2013年提出，全国人大层面的立法程序已经完成，大部分关键问题也已解决。他解释说：“最大难题在于确定计税依据。计税基础应是可自由交易的房产的市场价值，并给予一定折扣。但大量房产结构复杂且并非自由交易，难以确定合理的折扣。”

楼继伟指出，2022年被认为并非推出房地产税的合适时机。尽管房地产泡沫已破裂，但行业仍处在下行调整期，此时推出房地产税存在困难。他强调，房地产税仍应在“合适的时间”推出，这一点已写入十八届三中全会决定。

来源：新浪财经
