英伟达将为中国推出低价Blackwell AI芯片以应对美国出口限制

据知情人士透露，英伟达计划为中国市场推出一款价格显著低于此前受限的H20型号的新型人工智能芯片组，并最早于6月开始大规模生产。这款基于Blackwell架构的GPU预计售价在6500至8000美元之间，远低于H20的1万至1.2万美元，其较低价格反映了较弱的规格和更简单的制造要求。该芯片将采用常规GDDR7内存，而非更先进的高带宽内存，且不使用台积电的CoWoS封装技术。英伟达发言人表示，公司仍在评估“有限”选项，并需获得美国政府批准。由于美国出口限制，英伟达在中国的市场份额已从2022年前的95%骤降至目前的50%，其主要竞争对手为华为的昇腾910B芯片。

来源：路透社

SteamOS 3.7正式支持第三方AMD掌机及PC，华硕ROG Ally等设备获官方适配

Valve发布SteamOS 3.7版本，首次为Steam Deck以外的硬件提供官方支持，包括华硕ROG Ally、联想Legion Go系列等AMD掌机，并宣称搭载Ryzen Z1/Z2处理器的第三方设备可兼容基础功能。此次更新要求硬件必须为AMD架构并配备NVMe SSD，且需关闭Secure Boot进行安装。尽管SteamOS仍无法替代Windows（尤其缺乏对Intel/Nvidia硬件的支持），但Valve强调其已通过Steam Deck积累更稳定的游戏兼容性基础，成为游戏专用设备的可行选择。

四川屏山四川锦裕纺织有限公司5月20日发生纵火案。
对于网传纺织厂克扣、拖欠嫌疑人文某800元工资一事，屏山县公安局23日通报予以否认。通报称，文某办完离职手续要求支付工资时，工厂财务人员同意但要求按规定走流程。文某再次联系主管要求支付工资未果。文某20日在车间纵火并持刀刺伤财务人员。

美国总统国家安全事务助理迈克尔·华尔兹及其副手黄之瀚（Alex Wong）预计将于5月1日从白宫离职。1日下午，特朗普发帖宣布将提名华尔兹为美驻联合国大使（须获参议院确认），国务卿鲁比奥将临时兼任国安助理一职。 一名熟悉情况的人士称，特朗普认为Signal群组泄密事件已过去足够长的时间，两人离职可以被解释为重组的一部分。特朗普此前对解雇华尔兹持谨慎态度，不愿意被看作自己屈服于媒体和外界的压力。 （CBS）

特朗普正对白宫国家安全委员会进行重大改革，缩小其规模。
国安委主要负责在国家安全和外交政策方面为总统提供建议和协助，同时协调各政府机构。此次改革后，国安委的重点将是执行总统计划而非参与决策。预计国防部和国务院今后将在向特朗普提供建议方面发挥更重要的作用，但特朗普最终将“依靠自己的直觉”作出决策。
国安委内设部门将裁撤、合并，职能将转移至国务院、国防部等其他部门。消息人士称，原先负责处理非洲事务和处理北约等多边组织事务的部门将被合并。
目前约有395人在国安委工作，消息人士称预计最终可能会裁撤至50人左右。被裁人员中有约90人来自其他部门，系政策领域或某项事务的专家。被裁的职业公务员可以选择回到原部门，任命的官员也可以转至其他部门任职。
一名白宫官员称，重组国安委的目的还包括清除特朗普政府中的“深层政府”。特朗普认为这些人破坏了其美国优先的议程。此外，与特朗普关系密切的极右翼人士劳拉·卢默直接向他表达了对人员忠诚度的担忧。
（美联社，路透社，新华社）
