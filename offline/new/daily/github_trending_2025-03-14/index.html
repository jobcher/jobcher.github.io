外资持续撤离中国商业地产 四年累计净流出超百亿美元 供需失衡拖累市场复苏

受通缩压力加剧、全球融资成本上升及资本回报预期减弱影响，外资连续第四年净卖出中国商业地产。MSCI数据显示，2023年外资机构在工业资产以外的所有领域均呈净抛售，全年净流出9.69亿美元，2021-2024年累计净撤资达112亿美元，规模超过前两年净流入总和。其中办公楼资产遭抛售最甚，净流出达19亿美元，空置率攀升至23.5%且新增供应量同比激增70%。黑石集团2月将大湾区物流项目出售给平安保险，加拿大养老基金亦减持商业综合体股权，黑豹资本更因7.8亿元银团贷款违约将上海写字楼抵偿给渣打银行。尽管中国政府发行3000亿元超长期特别国债刺激消费，但世邦魏理仕预测，今年一线城市办公楼空置率或突破26.7%，零售物业空置率亦将持续走高，市场完全复苏仍需更长时间。
