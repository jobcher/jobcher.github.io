🖼 Cursor Pro 方案定价调整

Cursor Pro 方案定价调整

该公司承认，最近对 Pro 方案的定价变更沟通不清晰，对用户造成了困扰并损害了用户的信任。为此，他们表示歉意，并承诺为在过去三周内（6 月 16 日至 7 月 4 日）产生意外费用的用户提供退款。

Cursor 解释，其服务结合了自研模型以及来自 OpenAI、Anthropic、Google 和 xAI 等供应商的模型。过去，他们对外部模型的收费基于请求次数（每月 500 次），但新模型在处理复杂任务时，单次请求消耗的 token 数量可能远超简单请求。因此，基于 API 的实际使用量定价能更准确地反映成本。新的 20 美元额度，按额度池使用量计算，大约相当于 225 次 Sonnet 4 请求、550 次 Gemini 请求或 650 次 GPT 4.1 请求。

新 Pro 方案的内容

Auto 模式的无限使用：
在 Pro 方案中，用户可以在 Auto 模式下享受无限使用 Tab 和多种模型的权限。Auto 模式会根据容量自动切换不同的前沿模型，确保用户的使用需求得到充分满足。

每月 20 美元的前沿模型额度：
每个 Pro 方案用户将获得 20 美元的前沿模型使用额度，这些额度可用于调用包括 Sonnet、Gemini、GPT-4.1 等多种模型。根据模型的复杂度和任务的持续时间，所需的 tokens 数量可能会有所不同。

超出额度后的付费选项：
一旦超过 20 美元额度，用户仍可以按成本继续使用模型服务。为了避免意外超支，用户还可以启用消费上限功能，以控制额外的使用费用。

两名美国承包商的雇员受访并提供材料表明，守护援助物资分发点的承包商在巴勒斯坦人寻求援助时使用实弹、震撼弹和胡椒喷雾，对无辜的人造成了不必要的严重伤害。承包商Safe Reach Solutions证实采用实弹，但表示这是在高峰期控制人群的必要措施。
（美联社 1，2）
