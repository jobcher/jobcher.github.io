Steam Linux 平台占比升至 3.19%，AMD 成为绝对主导硬件

2025 年 12 月 Steam 硬件与软件调查显示，Linux 平台在 Steam 全球用户中的占比为 3.19%，较 11 月微降 0.01 个百分点，维持在历史高位区间。过去两年该比例持续上升：2023 年 12 月低于 2%，2024 年 12 月为 2.29%。考虑到年底假期和冬季特卖带来的整体活跃度增加，Linux 用户的实际规模可能保持稳定。

Linux 平台增长主要受 Steam Deck 带动。该设备运行基于 Arch Linux 的 SteamOS，并通过 Proton 兼容层支持大量 Windows 游戏，推动 Linux 游戏生态扩大。

硬件数据表明，AMD 在 Linux 平台的占比进一步上升。2025 年 12 月，AMD CPU 占比为 71.93%，环比增长约 5.2 个百分点；相比之下，Windows 平台中 AMD 占比为 47.27%。在 GPU 方面，“AMD Custom GPU 0405”（即 Steam Deck 所用 APU）占 Linux GPU 总量的 21.4%，据此推算约五分之一的 Linux 游戏用户来自 Steam Deck 设备。

🔍 开源推荐：witr - 追根溯源，一键搞懂“为什么这个进程在运行”

在服务器排查问题时，我们常使用 ps、top 或 lsof 查看系统状态，但这些工具通常只能告诉你“什么在运行”，却很难直观地解释“它为什么在运行”。

witr (Why Is This Running) 正是为了填补这一空白而生。它不仅支持通过进程名、PID 或端口号进行查询，核心功能更是将进程的因果链清晰地可视化出来。它能告诉你某个进程是由谁启动的（例如：systemd → pm2 → node），让你不再需要手动在多个工具间通过 PID 互相推导。
