洞庭湖紧急封堵失败 决堤口扩大至 226 米 水面落差0.1米

7月5日16时许，湖南省华容县团洲垸洞庭湖一线堤防发生管涌险情。17时48分许，紧急封堵失败后堤坝决堤。截至5日晚10时，缺口宽约8米、长约150米。目前，已安全转移安置群众5755人，未出现人员伤亡。

截止 7月6日上午，记者从湖南省岳阳市防汛抗旱指挥部获悉，截至6日11时，溃口处内外水位落差0.10米，长226米。

5日晚上，决堤口的宽度已延伸至约100米。6日凌晨左右，宽度已经延伸至约150米。根据当地水利部门介绍，当前如果强行进行堵口，难度很大。目前研究出来的方案是，等大堤两边的水位差稍微小一些或者持平的时候开始堵口。当前进行的工作是对团洲垸与其他垸之间的隔堤，组织人员力量加固防守。防汛设备物资连夜向现场运输。

记者从相关部门了解到此次溃口发生的过程：7月5日15点32分，团洲乡报险称团北堤段堤面发生塌陷；15点45分，华容县工程处险组到达现场，发现一处堤面下陷1.5米，堤身内坡有大量出水，流量约2立方米每秒。当地立即调集装满沙石车辆赶至现场进行紧急封堵。17时48分许，紧急封堵失败，堤坝决堤，经初步勘测，决堤口宽度约10米。

管涌溃口险情发生后，大量洞庭湖水涌进团洲垸，这里总面积55.03平方千米，2022年建成蓄洪安全区工程，安全区面积2.7平方千米、人口约3000人，蓄洪区外面积52.33平方千米、人口约5000人。当地调配县乡村干部700多人一户一户排查，紧急转移垸内居民至安全区内，第一时间对安全区外断电以确保安全。截至7月5日21点，附近6村1社区近5000人已完成转移，暂未出现人员伤亡情况。

消息来源: 央视新闻
