海地政府18日宣布成立临时选举委员会，为2016年以来的首次大选做准备。 委员会将代表农民、记者、人权活动人士以及宗教人士等团体，负责组织选举并帮助建立举行选举的法律框架。 过渡总统委员会成员史密斯·奥古斯丁称，按照法律规定，委员会应由9名成员组成，目前仅有7名成员，剩余2人将在未来几天内宣布。 （美联社）

海地过渡总统委员会10日解除加里·科尼耶的总理职务，将任命企业家阿利克斯·迪迪埃·菲斯-艾梅为新任总理。
过渡总统委员会成立于4月，任务是选择总理和内阁来平息动荡。但委员会一直饱受政治和内斗困扰，并与科尼耶不和。美洲国家组织据报上周曾试图调解分歧，但未能成功。
委员会3名成员10月面临腐败指控。调查人员声称他们向一名政府银行主管索要贿赂来保住其工作。这些指控可能会进一步削减民众对委员会的信任。
科尼耶近几周出访阿联酋和肯尼亚，试图寻求安全援助。在网传信件中，科尼耶表示委员会只有任命总理的权力而无法解职总理，委员会决议的行为超出了法律和宪法框架。
（美联社，路透社）
