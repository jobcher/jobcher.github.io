如图，消费、投资、出口都不及预期，GDP怎么会超预期？困惑ing~~

不要困惑，GDP统计方法有三种：生产法、支出法、收入法。生产法统计出来的GDP，不关消费和出口啥事，只有跟投资关系较为密切。像新能源车，只要生产出两台，就算卖一辆，拆一辆，或者一辆不卖放在停车场，只要生产了两台，就算两辆GDP。

但如果按支出法统计GDP，若生产岀来后没有消费者购买，就不能算入GDP。当然，还有一个方法是收入法，其中重要科目如劳动者报酬、营业盈余等，与生产法的统计科目也不交叉重叠。

我国对GDP的统计主要采用生产法，没法涵盖支出法中的最终消费支出，也不体现收入法中的劳动者报酬和营业盈余，体现不了消费不振和薪酬下调的真相。这也是为什么一些GDP统计数据与我们的体感存在温差的奥秘所在：统计数据未纳入我们的体感数据。
