甘肃省委调查组7月20日通报天水幼儿园血铅案情况：
2024年初，褐石培心幼儿园园长朱某琳为提高幼儿园招生吸引力，想通过增强食品色泽度和美观性，来吸引幼儿喜爱和家长认可，经投资人李某芳同意，授意厨师贺某霞购买颜料添加制作食品，持续拍摄照片视频在微信群和朋友圈传发。
其使用的黄色颜料铅含量209g/kg。从2024年5月至案发，平均每月6次向幼儿和教职员工提供食用。朱园长血铅亦超标。
该幼儿园曾购买食用色素，比颜料便宜，但使用比较后认为掺入颜料更鲜亮。
地表水、地下水、土壤样品送到北京检测，无污染。
案发前，共有7名幼儿来天水市第二人民医院就诊，发现血铅异常，检验科违规修改了其中2人的结果。对院长唐志锋、副院长周朝晖等5人立案审查。
案发后，甘肃省疾控中心7月2、3日检测时，样本静置时间过长、血浆分层，导致结果严重失真。后又通过多种方式逃避责任、阻碍调查。对中心党委书记王文军、主任孙建云、理化实验室主任李拥军立案审查。
麦积区教育局干部涉嫌接受李某芳利益输送。区、镇市监放弃值守。对教育局市监局局长等5人立案审查。
（甘肃省人民政府）

美国洛杉矶7月19日发生汽车撞人事件，造成30人受伤，其中7人伤势严重。

美国洛杉矶7月19日发生汽车撞人事件，造成30人受伤，其中7人伤势严重。
目击者称，出事前有一人被夜总会的人从里面轰出来，然后此人开着一辆银灰色的尼桑两厢小轿车冲向门口的人群，最后撞到一辆卖热狗的餐车。事发后围观者把司机从车里拉出来击打，混乱中还有人开枪射击。
警方确认，驾车司机受枪伤，已被送往当地医院。目击者称，枪手为一名身高一米七左右的男子，可能有一把银色的左轮手枪。事件中的受伤者多是当时等在夜店外的客人、代客泊车服务员和街边摆摊的小贩。
目前事发原因或动机尚不清楚。警方认为司机醉酒，没有迹象表明司机有其他犯罪意图或与恐怖主义有关。
（新华社，CNN）

↩️ 叙利亚当局努力实施停火，苏韦达省的冲突仍未完全停止。

竹新社:

叙利亚内政部与德鲁兹派宗教领袖谢赫·优素服·贾尔布赫7月16日宣布达成停火协议并立即生效。协议要求立即停止所有军事行动以缓和局势，同时成立由国家代表和宗教领袖组成的委员会来监督停火的实施。然而德鲁兹派另一名领袖谢赫·希克马特·希吉里拒绝接受停火，誓言继续战斗直至苏韦达省被全部解放。此前一天双方虽然达成停火协议，但很快被破坏。 以色列当天对叙利亚首都大马士革发动大规模空袭。以色列军方称其袭击了位于总统府附近的国防部总部，造成3人死亡、34人受伤。以军此前已对前往苏韦达省的政府军和车队发动数十次袭击。以军正在…

叙利亚当局努力实施停火，苏韦达省的冲突仍未完全停止。
过去一周，德鲁兹派和贝都因部落的武装人员一直在苏韦达省作战。叙利亚当局最初派去的军队被指站在贝都因部落一边对德鲁兹派发动袭击。以色列宣布支持德鲁兹派并袭击政府军。
叙利亚临时总统艾哈迈德·沙拉7月19日宣布停火，并开始在苏韦达省重新部署部队。停火要求保护德鲁兹人，并得到了以色列的批准。政府军设立了检查站，试图阻止更多人加入战斗。
当天晚些时候，叙内政部表示苏韦达市的冲突已经停止，当地所有贝都因部落武装人员都已离开。但记者仍在苏韦达市内听到枪声，并看到炮弹落在附近村庄。亦有武装人员抢劫商店并纵火。
（路透社，美联社，BBC）
