OpenAI 收购 Sky 开发商

2025年10月23日，OpenAI 宣布收购总部位于美国的 Software Applications Incorporated——这家公司是 macOS 自然语言界面工具 Sky 的开发者。

收购完成后，Sky 全体团队将加入 OpenAI。交易由 OpenAI 副总裁 Nick Turley 与 Fidji Simo 领导，并获得董事会独立委员会批准。

中国通过债务重组扩大人民币的全球影响力

中国通过债务重组扩大人民币的全球影响力

中国正利用其作为全球最大债权国的地位，通过放弃美元转而使用人民币，为海外借款人提供低国内利率的途径，以扩大全球人民币的采用。
埃塞俄比亚是最新考虑将其 53.8 亿美元债务的一部分转换为人民币贷款的国家，而更多主权国家正通过债券转向更便宜的中国融资。肯尼亚最近在将其中国铁路贷款从美元转换为人民币后，年度债务偿还成本削减了 2.15 亿美元。
分析师迈克尔·佩蒂斯表示，中国愿意接受较低的回报以换取更广泛的人民币使用。据熟悉政策的消息人士透露，北京支持将用于支付中国商品和服务的美元贷款进行转换，从而让借款人更容易以人民币结算贸易。该模式也可能适用于未来的贷款，范围将超越非洲。
尽管存在金融成本，这一策略加深了人民币在全球贸易和支付中的角色，对抗美国的金融霸权，同时增强了中国在非洲的影响力——自唐纳德·特朗普提高美国关税以来，非洲一直是关键的出口市场。
