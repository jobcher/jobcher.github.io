OpenRouter称GPT-5.5成本涨幅未达翻倍，但仍上升49%至92%

OpenRouter称GPT-5.5成本涨幅未达翻倍，但仍上升49%至92%

该平台基于从GPT-5.4转向GPT-5.5的用户群体进行对比，结果显示，GPT-5.5在实际使用中的成本增幅为49%至92%。OpenRouter认为，长提示词场景下，GPT-5.5生成内容更短，部分抵消了涨价影响，但整体成本仍明显上升。

OpenRouter在5月4日发布的成本分析中称，GPT-5.5相较GPT-5.4名义价格翻倍，输入价格从每百万token 2.50美元升至5.00美元，输出价格从每百万token 15美元升至30美元。

从输出长度看，超过1万token的提示词中，GPT-5.5生成的完成内容减少19%至34%；但在2千至1万token区间，完成长度反而增加52%。因此，短提示词用户面临的成本压力更明显。

英伟达CEO黄仁勋称人工智能正在创造大量就业机会

英伟达首席执行官黄仁勋在5月4日的米尔肯研究院对话中表示，人工智能是创造就业的工业级引擎，而非大规模失业的预兆。他认为AI为美国提供了“再工业化”的机遇，通过建设新型硬件工厂和基础设施，将带动全产业链的用工需求。黄仁勋强调，即便AI能够自动化处理特定任务，也不等同于取代整个岗位，员工的核心职能依然具有不可替代性。

针对行业内的“AI末日论”，黄仁勋批评此类言论过度引发公众恐慌，可能阻碍技术应用。他呼吁各界积极拥抱AI，以提升生产力并促进经济增长。尽管有学术机构预测AI可能导致部分职位流失，但黄仁勋坚持认为，AI带来的产业变革将最终转化为更广泛的就业增长和经济动力。
