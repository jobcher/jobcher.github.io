英伟达拒绝美国政府将“熔断开关”集成到出口中国的 AI 芯片中的计划；称此举将故意削弱关键基础设施，并表示这种事绝不会发生

2025年8月5日，NVIDIA对美国政府的提议表示明确反对，拒绝在出口至中国的AI芯片中集成“自毁开关”或后门程序。NVIDIA在声明中指出，此类措施不仅会削弱关键基础设施的安全性，还将为黑客和恶意行为者提供可乘之机，损害美国技术在全球市场的信任度。公司强调，其芯片设计始终坚持安全性原则，不会在任何设备中植入后门或自毁开关。

此前，NVIDIA曾因中国监管机构对其AI芯片安全性的审查而受到关注，公司此次专门发布博客文章，以回应相关质疑。NVIDIA还提及了历史上美国政府试图在加密通信中植入后门的“Clipper芯片事件”，认为类似措施将导致严重的安全漏洞，并对全球数字基础设施造成威胁。

尽管美国国会已提出相关法案，但NVIDIA明确表示，其芯片不存在任何后门、自毁开关或间谍软件，并称这种措施与构建可信赖系统的原则相悖。目前，美国政府尚未正式推进相关计划，但NVIDIA的立场已引发广泛讨论。

🖼 OpenAI 的开源模型发布

OpenAI 的开源模型发布

OpenAI 发布 gpt-oss：首批开放权重推理模型，最高 120 B 参数

OpenAI 8 月 5 日官方博客宣布推出两款开放权重语言模型 gpt-oss-120b 与 gpt-oss-20b，并采用 Apache 2.0 许可，这是公司自 GPT-2 以来首次开放权重的大模型。
gpt-oss-120b 在核心推理基准上接近 o4-mini，同样支持单张 80 GB GPU 推理。
gpt-oss-20b 表现与 o3-mini 相当，仅需 16 GB 内存即可在本地或边缘设备运行。
模型针对 agentic 工作流优化，原生支持指令跟随、工具调用（如网络检索与 Python 执行）、可调“推理力度”，并开放 完整 chain-of-thought 便于调试与可信度评估。
支持全参数微调，开发者可按低／中／高推理策略权衡成本与延迟。

两款模型经过系统化安全训练与评估；OpenAI 还对恶意微调后的 gpt-oss-120b 进行 Preparedness Framework 压力测试，确认其未达到高风险能力级别。

相关方法接受外部专家审阅，被视为开放模型安全标准的新进展。

国办印发《关于逐步推行免费学前教育的意见》

国办印发《关于逐步推行免费学前教育的意见》

日前，国务院办公厅印发《关于逐步推行免费学前教育的意见》，推进学前教育普及普惠安全优质发展。

意见明确，从2025年秋季学期起，免除公办幼儿园学前一年在园儿童保育教育费。免保育教育费标准按照县级以上地方人民政府及其教育、价格主管部门批准的公办幼儿园保育教育费收费标准（不含伙食费、住宿费、杂费等）执行。

对在教育部门批准设立的民办幼儿园就读的适龄儿童，参照当地同类型公办幼儿园免除水平，相应减免保育教育费。民办幼儿园保育教育费高出免除水平的部分，幼儿园可以按规定继续向在园儿童家庭收取。

对因免保育教育费导致幼儿园收入减少的部分，由财政部门综合考虑免保育教育费在园儿童人数、所在地保育教育费生均实际收费水平等情况补助幼儿园。

美国考虑通过芯片改造加强位置追踪以限制对华出口

据白宫科学技术办公室主任、上月特朗普总统公布的人工智能行动计划主要制定者之一迈克尔·克拉齐奥斯（Michael Kratsios）表示，美国正在评估对芯片进行潜在的软件和硬件改造，以提升位置追踪能力。

这一举措是华盛顿更广泛战略的一部分，旨在监控敏感半导体（例如英伟达生产的芯片）的流向，并限制它们进入中国。克拉齐奥斯在接受彭博电视采访时说：“目前正在讨论是否可以在芯片本身进行某种软件或物理层面的改造，以实现更好的位置追踪。”

美国计划与科技行业合作实施这些追踪措施，以防止走私并维护美国的技术优势。该问题已引发外交关注——上周，北京召见了英伟达代表，讨论美方在位置追踪以及与其 H20 芯片相关的其他所谓安全问题上的举措。

#机构点评 中国服务业7月意外走强，尽管消费者信心疲弱

根据标普全球PMI数据，中国7月服务业活动意外增强，指数从6月的50.6升至52.6，创下自2023年5月以来的最高水平。该读数超过经济学家预测的中位值50.4，显示在暑期旅游旺季期间实现了稳健扩张。

虽然夏季通常会带动旅游、交通和娱乐行业，但消费者信心依然疲软。央行的一项调查显示，二季度居民对就业市场的信心降至有记录以来最低水平。

与此形成对比的是，官方PMI数据显示7月小幅下降，从50.1降至50。国家统计局指出，交通运输、邮政服务和文化娱乐等行业实现了较大增长，但房地产及其他消费服务行业仍在收缩。
