OpenAI在ChatGPT上线年龄预测与18+验证

2026年1月20日，OpenAI宣布在ChatGPT面向个人用户的订阅计划中逐步上线“年龄预测”，用于判断账号是否可能属于18岁以下用户，并自动切换到更严格的青少年保护体验。年龄预测模型基于多种行为与账号层信号进行估计，包括账号存在时长、常见活跃时段、长期使用模式以及用户自报年龄；当年龄信息不完整或不够确定时，若被判定可能未满18岁，ChatGPT会启用额外安全设置，降低对敏感内容的暴露，覆盖血腥暴力、可能诱导未成年人冒险的“挑战”内容、性/恋爱/暴力角色扮演、自残相关内容，以及鼓吹极端审美或不健康节食、羞辱身体等内容。

对被误判为未成年但实际已满18岁的用户，平台提供年龄验证以恢复完整访问：通过Persona进行自拍验证（部分国家/地区可能还需政府签发证件）。OpenAI称不会收到用户的证件或自拍，仅接收出生日期或18+验证结果；Persona会在验证后7天内删除证件/自拍。用户也可通过完成验证来停止在其账号上继续运行年龄预测。

该功能被描述为全球滚动上线，欧盟因地区要求将在“未来几周”推出；用户可在设置中查看是否处于“under-18 mode”，并随时发起验证流程。
