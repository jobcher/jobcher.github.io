天选打工人！研究显示亚洲人更能忍耐工作

来自荷兰拉德堡德大学的研究人员在国际权威心理学期刊《心理公报》（Psychol Bull）发表的文章中，研究人员筛选确定了170项研究，涉及358种不同任务和4670名受试者，涵盖了医疗工作者、军事人员、业余运动员、大学生等人群，受试者来自全球29个不同的国家。
研究结果发现，无论是重复的任务、有反馈或没有反馈的任务、高控制或低控制的任务、高意义或低意义的任务，动脑都“一视同仁”地让我们感到不愉快。也就是说，当人们需要付出更多脑力劳动来完成任务时，他们更容易感到沮丧和不满。唯一的例外是，相较于欧洲和北美地区，亚洲地区的研究显示，脑力劳动带来的不快感稍低。
