Anthropic 发布 Claude Opus 4.7

Anthropic 发布 Claude Opus 4.7

Anthropic 于 2026 年 4 月 16 日发布 Claude Opus 4.7，并将其描述为目前能力最强的公开可用 Opus 型号。官方称，新版本相较 Opus 4.6 在高级软件工程任务上有明显提升，尤其强化了长时任务处理、严格按指令执行，以及在返回结果前进行自我校验的能力。

Claude Opus 4.7 已上线 Claude 全系产品、Anthropic API、Amazon Bedrock、Google Vertex AI 和 Microsoft Foundry，API 标识为 claude-opus-4-7。价格保持不变，仍为每百万输入 token 5 美元、每百万输出 token 25 美元。Anthropic 也已在模型文档中将其列为复杂推理与 agentic coding 场景的优先推荐型号。

安全层面，Anthropic 表示 Opus 4.7 是其首个用于测试新网络安全防护机制的模型，系统会自动识别并拦截涉及禁止或高风险网络安全用途的请求；对于漏洞研究、渗透测试和红队等正当用途，官方同步开放了 Cyber Verification Program。与此同时，开发者文档显示，旧版 Claude Opus 4 已于 2026 年 4 月 14 日被弃用，官方推荐迁移至 Opus 4.7。

Qwen 发布 Qwen3.6-35B-A3B 开放权重模型

Qwen 发布 Qwen3.6-35B-A3B 开放权重模型

阿里通义今日宣布推出 Qwen3.6-35B-A3B，并以 Apache 2.0 许可发布。该模型采用稀疏 MoE 架构，总参数为 350 亿，推理时激活参数为 30 亿，是 Qwen3.6 系列首个开放权重版本，目前已上线 Hugging Face；官方同时预告，托管 API 版本 “Qwen3.6-Flash” 将登陆 Model Studio。

从官方模型卡看，Qwen3.6-35B-A3B 继续把重点放在代码与真实开发流程上，强化了前端工作流处理、仓库级推理，以及多轮开发中对 reasoning context 的保留能力。模型支持图文输入，官方还强调其支持 multimodal thinking 与 non-thinking 两种模式；原生上下文长度为 262,144 token，并可扩展至约 101 万 token。

按 Qwen 公布的基准结果，这一版本在 SWE-bench Verified、Terminal-Bench 2.0 等 agentic coding 指标上，相比前代 Qwen3.5-35B-A3B 有明显提升，在 RefCOCO 和 ODInW13 等视觉与空间理解任务上也进一步增长。

考科蓝合作组织称阿尔茨海默病“特效药”缺乏临床实效

考科蓝合作组织（Cochrane）发布的一项重大评估研究指出，针对淀粉样蛋白的阿尔茨海默病药物（如Lecanemab和Donanemab）在实际临床中并无显著益处。研究分析了涉及2万多名患者的17项试验，结果显示，虽然此类药物能清除脑部淀粉样蛋白，但并未有效改善患者的生活质量或减缓认知衰退。
此外，评估还强调了该类药物可能引发脑肿胀、出血等严重副作用。尽管部分药物此前已获多国监管机构批准，但研究人员提醒需警惕“过度乐观”导致的误导，呼吁科学界探索除淀粉样蛋白假说之外的其他治疗路径。目前，英国相关医疗监管机构已着手对该评估结果进行分析。

延长手机使用寿命的最佳方式是更换电池。

其实很简单。

用热风枪或吹风机对手机背部进行加热。

用剃须刀片、美工刀或其他锋利的小刀，小心地沿胶水处切割，即可轻松取下后盖。

可用拉片或（万不得已时）少量异丙醇来去除固定电池的胶粘剂。

使用简易卡扣式连接器，轻松更换电池。

将手机粘合复原。
