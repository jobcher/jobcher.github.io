OpenCode 按法律要求清理 Anthropic 内置接入

OpenCode 按法律要求清理 Anthropic 内置接入

3月19日，OpenCode 在 GitHub 合并了 PR #18186，标题为 anthropic legal requests。PR 发起说明写道，此次调整是“按法律要求移除 Anthropic 相关引用”，并点名包括删除 Anthropic 专用提示词文件、移除 provider hints 中的 Anthropic 提示，以及移除内置的 opencode-anthropic-auth 插件。

从公开提交内容看，anthropic-20250930.txt 已被整文件删除，opencode-anthropic-auth@0.0.13 已从内置插件列表移除，anthropic-beta 请求头里的 claude-code-20250219 标记也被删去，CLI 登录界面中原先面向 Anthropic 的 API key提示标签同步移除。
