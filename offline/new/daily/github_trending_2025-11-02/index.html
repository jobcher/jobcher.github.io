中国正在取消一项长期实行的黄金税收优惠政策，这可能会对全球顶级黄金市场之一的消费者造成打击。

中国正在取消一项长期实行的黄金税收优惠政策，这可能会对全球顶级黄金市场之一的消费者造成打击。

根据财政部的一项新法规，从11月1日起，北京将不再允许零售商在出售从上海黄金交易所购买的黄金时抵扣增值税，无论黄金是直接出售还是经过加工后出售。

该规定涵盖投资产品（如高纯度金条、金锭以及中国人民银行批准的金币）和非投资用途（包括珠宝和工业材料）。

此举有望在房地产市场低迷和经济增长乏力导致财政收入紧张之际，提振政府财政收入。但这些变化也可能增加中国消费者购买黄金的成本。

近期全球散户投资者的疯狂购买热潮推动黄金价格创下历史新高，并进入超买区域，这为贵金属价格的突然回调埋下了伏笔。
