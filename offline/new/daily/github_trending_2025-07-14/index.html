特朗普7月12日宣布将自8月1日起对欧盟和墨西哥商品征收30%关税。
特朗普宣称，墨西哥未能阻止芬太尼等进入美国，在配合美国阻止非法移民进入美国方面也做得不够；欧盟的关税和非关税贸易壁垒导致美国出现对欧巨额贸易逆差，双方关系“远非互惠互利”。
信中措辞与过去一周发给其他国家领导人的信基本一致，包括警告对方不要对此次加税进行报复、鼓励对方企业迁入美国、如对方积极配合可能对税率进行调整等。
欧盟委员会主席冯德莱恩回应说，欧盟将采取一切必要措施维护自身利益，包括在必要时对等反制。但欧盟始终寻求与美国通过谈判解决问题，仍然准备在8月1日之前继续努力达成协议。欧洲多国亦表示强烈反对。
墨西哥政府发布公告说，美方正式提出的对墨关税举措“不公平”，已就保护企业与就业岗位展开对美谈判。墨总统克劳迪娅·辛鲍姆表示将寻求外交方式解决分歧，同时不会在国家主权问题上作出任何让步。
