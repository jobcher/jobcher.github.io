怎样的“反内卷”才能提振总需求？——从罗斯福到凯恩斯

怎样的“反内卷”才能提振总需求？——从罗斯福到凯恩斯

即便我们相信理论是正确的，当前的中国经济也并不满足“反内卷”提振总需求的三个前提条件（零利率下限+通缩螺旋+快速去杠杆）。
﻿
更重要的是，目前的“反内卷”相较于NIRA欠缺了最重要的一个组成部分——提高居民收入。NIRA的核心特征之一是通过劳工政策直接干预收入分配，通过强化劳工保护、提高工资水平来提振收入预期。相比之下，当前中国的"反内卷"主要聚焦于限制价格竞争、规范市场秩序，在提高居民收入方面着力有限。

当前我国“反内卷”政策既不具备生效的极端宏观条件，也缺乏NIRA式收入再分配的机制，其本身可能并不足以推动总需求持续改善。
