美国联邦通信委员会扩大对部分中国科技产品的进口禁令

美国联邦通信委员会（FCC）于当地时间6月26日宣布，将进一步禁止进口部分中国制造的电子设备。该决定扩大了2022年实施的禁令范围，不仅涵盖华为、中兴、海能达、海康威视和大华股份的新型通信及视频监控设备，还将范围延伸至用于公共安全和关键基础设施监控等领域的旧款模型。

此次扩大的禁令预计将于7月初生效，旨在防范所谓的国家安全风险。不过，FCC表示美国民众仍可继续使用已持有的旧设备。此外，FCC目前还在考虑禁止美国电信运营商与中国电信企业互联，这或将影响中国电信企业在美数据中心运营。

来源：路透社

OpenAI 开启 GPT-5.6 有限预览

OpenAI 开启 GPT-5.6 有限预览

OpenAI 于 2026 年 6 月 26 日宣布开始 GPT-5.6 系列有限预览。新系列包括旗舰模型 Sol、面向日常工作的 Terra，以及低成本模型 Luna。官方称 Terra 的性能接近 GPT-5.5，但价格降低一半；Luna 则主打更低成本。此次并非全面开放，而是先通过 API 和 Codex 面向少数受信任合作伙伴及机构提供。

价格方面，GPT-5.6 按三档模型计费：Sol 为每百万输入 token 5 美元、输出 token 30 美元；Terra 为输入 2.5 美元、输出 15 美元；Luna 为输入 1 美元、输出 6 美元。OpenAI 还表示，GPT-5.6 Sol 将于 7 月在 Cerebras 上线，速度最高可达每秒 750 token，但初期仍仅限部分客户使用。

长江存储全球NAND市场份额增至13% 与闪迪并列第五

据Counterpoint Research最新报告显示，中国NAND闪存制造商长江存储（YMTC）在全球市场的份额已从2025年同期的8%大幅激增至13%，与闪迪（SanDisk）并列全球第五，并直逼第四名的日本铠侠。受内存短缺和价格上涨推动，长江存储在2026年第一季度营收达26亿美元，同比飙升近445%，成为全球NAND市场增长最快的企业。

行业分析指出，长江存储在技术和产能上的快速扩张已对三星、SK海力士等韩国传统巨头构成竞争压力。此外，市场预期长江存储近期可能紧随长鑫存储（CXMT）之后进行IPO。另有市场消息称，长江存储此前已接受苹果公司的芯片验证测试，有望进入其供应链，这进一步引发了国际同行对其市场颠覆能力的关注。

美国和伊朗再次相互袭击。
美军中央司令部指责伊朗6月27日早些时候袭击悬挂巴拿马国旗的油轮，美国战机对伊朗的军事监视、通信、防空、无人机储存以及布雷设施等霍尔木兹海峡及其附近多地的10个军事目标进行打击。伊朗南部锡里克地区传出爆炸声。
巴林27日早些时候发表声明，指责一些伊朗无人机瞄准该国，袭击是对该国公民和居民安全的公然威胁。
伊朗革命卫队28日说，革命卫队海军和空军用导弹和无人机，摧毁了位于科威特阿里·萨利姆空军基地和位于巴林的美军第五舰队的8处美军重要基础设施，对近日美国的“侵略”行径作出了果断回应。
特朗普威胁，总有一天，美国将无法保持理智，而被迫以军事方式完成行动。伊朗革命卫队威胁，美军在中东地区的军事基地将在这几天体验到“地狱”般的感受。
伊朗尚未直接评论针对船只袭击的报道。 但伊朗媒体报道称，革命卫队向试图通过未经伊朗批准的航道的不明船只发出了“警告”，以促使其他船只寻求伊朗的许可。
（路透社，美联社，新华社 1，2）
