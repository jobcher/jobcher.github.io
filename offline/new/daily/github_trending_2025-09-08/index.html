韩国已与美国完成释放韩国工人的谈判

韩国已与美国完成释放韩国工人的谈判

韩国总统府官员周日表示，韩国已与美国完成谈判，以促成在佐治亚州被拘留的韩国工人获释。
总统府幕僚长姜勋植在电视讲话中称，一旦行政程序完成，将派遣飞机接回这些韩国公民。此前，美国联邦探员对佐治亚州一家现代汽车制造工厂进行了突击检查，拘留了数百人，其中大部分为韩国公民。此次行动是美国国土安全部调查行动史上规模最大的单一场所执法行动。
韩国政府表示将寻求改善韩国工人赴美签证制度，以“防止类似事件”再次发生。

来源：路透社

尽管中国的生产者价格指数（PPI）同比增速仍处负值区间，但高频数据显示，在“反内卷”政策推动下，部分工业品于7月下旬开始，已出现减产和价格回升。具体案例包括钢铁减产、宁德时代暂停旗下锂矿的运营，以及农业部计划削减100万头能繁母猪（约占存栏总量的2.5%）以提振猪价。受此影响，预计生产者价格指数的同比降幅将在2025年底收窄至2%以内，并有望在2026年转为正增长。

日本首相石破茂宣布辞去自民党总裁职务。
自民党在参院选举大败后，包括内阁成员在内的该党国会议员和地方组织中要求提前举行总裁选举的呼声扩大，石破茂丧失凝聚力，放弃了继续任职的念头。
石破茂认为，日美关税谈判告一段落的现在正是恰当的辞职时机。他明确表示不会参加下届总裁选举。关于7月参院选举大败一事，石破茂称“责任在于我这个总裁”。
自民党计划根据8日提交的要求提前实施选举的该党国会议员的书面文件，结合各都道府县支部联合会的要求，公布是否提前举行总裁选举。石破茂将在新总裁、首相就任之际辞职。
被视为“后石破”的人选包括，去年总裁选举中曾与石破在最终投票中对决的前经济安全保障担当相高市早苗、农相小泉进次郎、前经济安保相小林鹰之、官房长官林芳正等人。
（NHK，共同社）
