四部门：营造良好的货币金融环境

国家发展改革委办公厅等发布关于做好2024年降成本重点工作的通知。其中提到，营造良好的货币金融环境。稳健的货币政策要灵活适度、精准有效，综合运用多种货币政策工具，保持流动性合理充裕，社会融资规模、货币供应量同经济增长和价格水平预期目标相匹配。畅通货币政策传导机制，避免资金沉淀空转。保持人民币汇率在合理均衡水平上的基本稳定。
