波音公司“星际客机”预计9月初不载人返回地球。这能让航天局和波音在其返回过程中继续收集相关测试数据，同时不让宇航员“承担超过不必要范围的风险”。
航天局官员均支持“龙”飞船将宇航员带回地球。“星际客机”表示将采取必要行动支持航天局决定。不过航天局并未放弃波音，局长比尔·纳尔逊表示“星际客机”将再次飞行。
航天局飞行运营局副局长诺姆·奈特称其24日与滞留宇航员威尔莫尔和威廉姆斯进行交谈，他们完全支持推迟返回的决定。宇航员此前受访时表示，他们没有任何抱怨，喜欢参与在空间站的工作。
“星际客机”6月5日发射升空，原定6月14日脱离空间站返回地球。但“星际客机”出现推进器故障和氦气泄漏等一系列问题，导致返航时间一再推迟，两名宇航员至今已滞留国际空间站79天。
波音在地面安排测试和模拟，试图证明可以将宇航员带回。但测试结果提出了更困难的工程问题，未能解决航天局对推进器及其载人能力的担忧。
（路透社，美联社，新华社）
