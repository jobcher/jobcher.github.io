Linux 内核迎来多内核架构提案

Linux 内核迎来多内核架构提案

9 月 21 日消息，Multikernel Technologies 向 Linux 内核邮件列表提交了一份 多内核架构（Multikernel Architecture） 的 RFC 补丁，并已开源相关代码。该方案允许多个独立的内核实例在同一物理机上并存运行，每个内核可绑定特定 CPU 核心，同时共享底层硬件资源。

开发者 Cong Wang 表示，多内核架构的优势包括：

更强的故障隔离与安全性；

高于传统虚拟机的资源利用率；

通过 KHO（Kernel Hand Over） 实现潜在的零停机内核更新；

支持实时内核与通用内核并行运行，适配安全关键或专用任务。

该实现基于 kexec 基础设施 管理多个内核镜像，并通过专用的 IPI 框架实现跨内核通信。

目前社区已有开发者表示，曾在 x86 硬件上尝试过类似概念，但要突破实验阶段仍面临不少复杂性。Linux 上游是否采纳这一提案，还有待观察。

山东临沂农妇杨宝花因“辱骂”法官，被判行政拘留15天，罚款10万元。
2023年，杨宝花的丈夫被同村村民手持羊角锤打伤，2025年6月3日，临沂经开区法院判处凶手拘役三个月，赔偿2.5万元。
判决下达后，杨宝花质问承办法官“为什么（对被告）判得这么轻？如果是你的丈夫被别人打成这样，你也会这样判吗？”，“谁这么判决，谁就没有良心”。
随后，杨宝花被法警带走，当天，临沂经开区法院对杨宝花作出罚款决定书和拘留决定书，认定杨宝花对工作人员进行侮辱、谩骂，态度十分恶劣，其行为已严重妨碍人民法院正常的刑事诉讼活动。对杨宝花罚款10万元，同时对其拘留 15 日。
拘留结束后，杨宝花向临沂中院申请复议。但临沂中院工作人员回复称，收到相关决定书后，三日内可以申请复议，杨宝花的复议申请超期。
9月19日，临沂经开区人民法院发布情况通报，撤销罚款决定，将罚款及利息退还，但并未承认拘留错误。
（中国新闻周刊，临沂经开区人民法院）
