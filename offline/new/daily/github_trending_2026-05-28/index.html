日本众议院26日通过了《个人情报法》修正案。
修正案将松绑对统计编制和AI开发等无法识别个人身份用途的限制。允许企业可不经本人同意便能收集通过社交媒体等公开来源的信息，以及与其他公司共享所持有的数据。
相关信息除疾病和犯罪记录外，还包括种族和信仰等。这些信息被列为“需特别注意的个人信息”，根据现行规定，原则上必须征得本人同意方可获取。但由于研发AI需大规模收集互联网上的信息，因此难以逐一获取个人的同意。
罚款制度针对非法获取、利用超过千人份大规模个人信息的商家等，责令其缴纳相当于所获利益相当额的罚款。
在野党议员对新法案可能造成敏感信息的泄露表示担忧。
（共同社）
