消防反腐持续发力。9月27日，中央纪委国家监委网站通报，四川省森林消防总队党委副书记、总队长夏进荣涉嫌严重违纪违法，主动投案，目前正接受中央纪委国家监委驻应急管理部纪检监察组纪律审查和福建省龙岩市监察委员会监察调查。

据四川省应急管理厅官网，夏进荣还是四川省应急管理厅党委委员。他的职责分工是，负责全省森林草原消防队伍建设和队员教育训练、森林草原消防救援基地建设，受厅党委委派组织、参与、指导森林草原火灾扑救和抢险救援等相关工作；分管火灾防治管理处。承担厅党委议定的专项工作；完成党委书记、厅长交办的其他工作任务。

值得注意的是，夏进荣投案之前，他的老领导、云南省森林消防总队原党委副书记、总队长齐兴彬已先一步投案。在齐兴彬任内，夏进荣担任云南省森林消防总队参谋长。2025年6月30日，中央纪委国家监委网站通报齐兴彬涉嫌严重违纪违法，主动投案，目前正接受中央纪委国家监委驻应急管理部纪检监察组纪律审查和云南省监委监察调查。齐兴彬官宣被查时已退休一年多。
