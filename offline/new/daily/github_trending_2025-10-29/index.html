微软与 OpenAI 签署新协议，重塑长期合作关系

微软与 OpenAI 签署新协议，重塑长期合作关系

2025年10月28日，微软宣布与 OpenAI 签署新的正式协议，进入合作伙伴关系的下一阶段。微软支持 OpenAI 组建公益型公司（Public Benefit Corporation, PBC）并完成资本重组。重组后，微软持有约1350亿美元的股份，折合约27%的稀释后股权比例（此前为32.5%）。

协议延续了双方在前沿模型和 Azure API 独家权的核心合作框架，直至 OpenAI 宣布实现通用人工智能（AGI）。AGI 的宣布将由独立专家小组验证。微软的模型和产品相关知识产权延长至2032年，并覆盖AGI之后的新模型；研究类IP权利保留至2030年或AGI确认前。微软的独家权不再包括 OpenAI 的消费硬件。

新条款允许 OpenAI 与第三方联合开发部分产品，其中 API 类产品仍专属 Azure。微软也可独立推进 AGI 研发。双方的收益分成机制维持至AGI确认，但付款周期延长。OpenAI 同时签约新增采购2500亿美元的 Azure 服务，微软不再拥有计算资源优先权。

此外，OpenAI 获准向美国国家安全部门提供API服务，并可在符合能力标准的情况下发布开放权重模型。

这一调整标志着微软与 OpenAI 的合作从战略绑定走向结构性分化，两家公司将在保持互信的前提下，获得更大自主空间以继续推动人工智能的发展。

来源：Microsoft 官方博客

四十五五的《建议》稿，最大的看头就是目标这一段，其他都是围绕这段展开：

1、经济发展：发展→高质量发展。
更加注重稳增长：“持续健康发展”→“增长保持在合理区间”
新增全要素生产率目标，量化新质生产力；
新增消费率、内需拉动等目标，这是供需再平衡、做强“内循环”的量化目标；
“四化”目标中，优先级次序很明显：新型工业化＞信息化＞城镇化＞农业现代化；

2、新增“科技自立自强”目标，要求“大幅提高”，并进一步量化为几个子目标(见图)

3、改革目标，
放在首位的是“国家治理”，市场经济体制退而其次，更加强调政府有形之手，而非市场无形之手，这和三中全会定调是一致的。

4、社会文明目标，
“文化自信”放在了首位，其次是“主流思想舆论”(俗称“正能量”)，然后也新增了“文化创新创造活力”，以及“文化影响力”目标。

5、民生目标，
首先，民生目标的总体定调比较底，仅是“不断提高”，其他目标基本都有“明显”、“显著”、“大幅”等定性词。
其次，对于居民收入和经济增长的匹配，定调比较积极，去掉了“基本”一词，意味着收入增长要追赶经济增长，直至二者“匹配”。
同时，新增了劳动报酬和劳动生产率匹配的目标，
这个目标应该是蔡昉等人极力建议的，主要避免如AI等新质生产力发展过快，对劳动力市场的破坏。若新质生产力发展过快，就业岗位的淘汰可能大幅快于新增就业，从而劳动报酬增长可能大幅落后于生产率增长，可能导致新质生产力越火热，人民生活越困难的割裂局面。
然后，对于“分配”的目标，定调较低，仅仅是“得到优化”，而十四五都是“明显改善”，难道说已经明显改善了吗？但同时新增了“中等收入群体”的目标，要求“持续扩大”。
最后，对于社保目标，定调的是“更加优化更可持续”，这和民生定调“尽力而为、量力而行”是一样的，依然是很保守的定调。

6、美丽中国目标，最重要的子目标，除了碳达峰，就是“新型能源体系”，这也是自立自强的重点。

7、新增了“国家安全”目标，防风险优先级更高，如重点领域风险，社会稳定风险等等。这些之前简报前瞻中就聊过，就不再赘述。

8、2035远景目标又正式迎来更新，新增了“国际影响力”目标，人民生活新增“幸福”。

来源：宏观边际

如何理解央行今天金融街的讲话？
1、央行将进一步强化宏观审慎管理，这是上周五央行学习四中全会时，列举的“十五五”央行重点内容之一；

2、这其实是将过去央行“最后贷款人”职能，扩展至“最后做市商”，虽然目前央行没有明确提及或承认“最后做市商”的说法，但已经实质性的朝着这个方向在转变；

3、通俗点讲，就是过去央行只负责解决金融机构流动性不足的问题，现在央行还要解决“资产”流动性的问题，即充当被抛售资产的最后买家。
再通俗点讲，就是央行将更加直接或间接地干预股市、债市、汇市等金融市场，甚至也包括楼市。
比如今年4月份贸易战爆发时，为了稳住股市，推出的“类平准基金”，央行直接给中央汇金提供流动性；
再比如，去年创设的国债买卖工具，央行在二级市场买卖国债，直接干预债市；

4、这意味着，以后股市“类平准基金”将常态化，将进化为真正的“平准基金”，由央行主要提供子弹。
必要时，央行甚至可以考虑直接下场买，这明确给股市上了“保险”，救市也将比以前更加迅速、果断且有效。
