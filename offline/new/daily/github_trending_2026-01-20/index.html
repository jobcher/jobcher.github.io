补丁版 Wine 跑通 Adobe Creative Cloud 安装器

补丁版 Wine 跑通 Adobe Creative Cloud 安装器

开源开发者 PhialsBasement 解决了 Adobe Creative Cloud（Windows）安装器在 Wine 上的关键兼容性问题，使 Photoshop 2021 与 Photoshop 2025 能在 Linux 上完成安装与启动（基于未合并的补丁）。补丁主要围绕 Wine 的 mshtml 与 msxml3：修复 JavaScript 分发/事件处理、以及 XML 解析的兼容性（包括对安装器生成的“非严格 XML”做预处理，避免 libxml2 解析失败；PR 描述中提到可修复安装进度约 30% 处的 E103 错误）

在应用运行层面，开发者反馈 Photoshop 2021 “运行很顺滑”，但拖拽（drag-and-drop）存在问题，怀疑与 Wayland 环境有关。补丁最初以 PR 形式提交到 ValveSoftware/wine（Proton 相关的下游 Wine 仓库），但维护者明确要求先向 WineHQ 上游提交并合并，因此该 PR 已被关闭；也意味着短期内不会直接进入 Proton/Wine 的主线发布。

开发者同时提供了一个 GitHub 仓库/Release（fix-adobe）供有需求的人自行测试未合并代码

对于特朗普的关税威胁，欧洲国家纷纷予以谴责。他们重申支持丹麦，认为关税不应成为格陵兰岛问题讨论的一部分。关税威胁还可能破坏特朗普此前与欧盟和英国达成的初步协议。
有欧洲外交官质疑美国如何对欧盟经济区中的部分国家实施关税，此外，特朗普的关税并未给出法律依据。
丹麦和格陵兰岛17日爆发抗议活动，反对特朗普的威胁，呼吁让格陵兰自行决定未来。
