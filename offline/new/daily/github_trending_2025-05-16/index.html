谷歌为Android和Chrome推出全新AI及无障碍功能

谷歌5月15日宣布为Android和Chrome推出多项AI及无障碍升级。其中，Android屏幕朗读工具TalkBack新增集成Gemini功能，视障用户可直接询问图片内容（如朋友发送的吉他照片的品牌、颜色）或当前屏幕信息（如购物APP中商品的材质、折扣）。去年已实现的AI图片描述功能现扩展至交互式问答。

此外，Android实时字幕功能"Expressive Captions"新增语音拖长音识别（如"amaaazing"）和环境音标签（口哨声、清嗓声），目前面向美、英、加、澳的Android 15+设备开放英语版本。

Chrome方面，桌面版首次支持通过OCR技术朗读扫描版PDF文本（可高亮、复制、搜索）；Android版新增"页面缩放"功能，可单独调整文字大小而不影响网页布局，设置路径为浏览器右上角三点菜单。

🖼 CS饰品市场跌入谷底，绝大部分道具价值在15天内近乎腰斩

CS饰品市场跌入谷底，绝大部分道具价值在15天内近乎腰斩

在经历了5月6日创纪录的40%涨幅后，《反恐精英》游戏饰品市场迅速陷入自由落体状态，仅仅一天后就出现了20%的急剧回调，引发了持续的市场恐慌情绪。

随着恐慌情绪蔓延，持有大量库存的玩家纷纷加入抛售行列，进一步加剧了市场的下行压力。据业内人士透露，CS饰品交易生态系统中的"商人"群体，长期扮演着类似金融市场做市商的角色，为场外玩家提供库存托管和交易渠道，实质上创造了一个类似场外衍生品的灰色市场。

然而，这一脆弱的市场结构在7日暴跌后迅速崩塌。知情人士表示，这些托管商为了对冲自身损失，不得不大规模清仓，导致市场供给激增，形成了恶性循环。

市场分析显示，截至目前，除了少数磨损度极低（即品相优良）的高端饰品出现些许企稳迹象外，绝大多数道具仍在延续下跌走势，市场信心尚未恢复。
