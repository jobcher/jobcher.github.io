核聚变：EAST论文报告进入“密度自由区”，线平均密度达1.65倍Greenwald极限

核聚变：EAST论文报告进入“密度自由区”，线平均密度达1.65倍Greenwald极限

2026年1月1日，《Science Advances》发表同行评审论文，报告在中国EAST（Experimental Advanced Superconducting Tokamak）上，通过ECRH（电子回旋共振加热）辅助欧姆启动，实现线平均电子密度达到Greenwald经验密度极限的1.3–1.65倍（论文同时给出EAST常规运行约0.8–1.0倍），并将相关放电归入等离子体-壁自组织（PWSO）理论预测的“密度自由区”工况。

论文把“密度极限”定义为放电崩塌前瞬间达到的最大线平均电子密度；其操作路径是提高启动前的预充/初始中性气体密度，并在启动阶段施加ECRH，同时使用密度反馈控制把放电推向高密度。论文给出代表性放电参数与测量定义，包括线平均密度由干涉仪测得；示例放电包含约2.5 T磁场、约250 kA等离子体电流、ECRH约600 kW等量级条件。

在机理解释上，论文将实验数据与PWSO理论的0D与1D模型进行定量比较，认为这些放电位于PWSO所述的“density-free basin”。论文指出，在EAST全金属壁、钨等高Z等离子体面对部件条件下，随着ECRH功率和/或预充压力提高，偏滤器靶板附近温度降低、平顶阶段可达密度极限上移；并提出这指向一种在不主动注入杂质的情况下，通过获得脱离（detachment）来显著抬升托卡马克可运行密度上限的可操作方案。

Science Advances（PMC全文）：Jiaxing Liu 等，“Accessing the density-free regime with ECRH-assisted ohmic start-up on EAST”，2026-01-01，DOI:10.1126/sciadv.adz3040。

OpenAI 表示，基于其匿名化对话分析，全球每周有超过 2.3 亿人会在 ChatGPT 提出健康与保健相关问题；Health 允许用户在授权后连接医疗记录与健康应用（如 Apple Health、Function、MyFitnessPal 等），以便围绕化验结果解读、就诊提问准备、饮食与运动建议、保险方案取舍等任务获得更贴合个人情境的回复。

Radeon RADV Vulkan 驱动接近实现新一轮光线追踪性能提升

Valve Linux 图形团队的开源开发者 Natalie Vock 近期提交了一项 Mesa 合并请求，对 AMD Radeon 的 RADV Vulkan 光线追踪代码进行关键调整。该修改主要重构了 launch ID swizzling 逻辑，用于修复《Unreal Engine 5》中 Lumen 全局光照与反射系统在 Linux 下的运行问题。

根据合并请求中的说明，此次调整在 Unreal Engine 5 的硬件 Lumen 渲染通道中可带来约 4 至 8 倍的性能提升，整体帧率在初步测试中提升约 30%。相关测试主要针对通过 Steam Play 运行的 Linux 游戏环境。

在社区讨论中，部分用户还分享了其他游戏的基准测试结果，显示 RADV 光线追踪性能在多个场景下均有明显改善，一些此前表现卡顿的游戏在更新后达到可流畅运行水平。

该合并请求目前仍处于审核阶段，若进展顺利，预计将被纳入 Mesa 26.0 稳定版发布周期。Mesa 26.0 计划于 2026 年 2 月发布，届时相关优化将正式进入主流 Linux 图形栈。
