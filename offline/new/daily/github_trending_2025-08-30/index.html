🖼 Mihomo 内核项目或将停止开发，下游 GUI 项目拒绝改名引发危机

Mihomo 内核项目或将停止开发，下游 GUI 项目拒绝改名引发危机

开源代理内核 Mihomo 开发团队今日表示，正在认真评估是否停止内核开发，以规避因下游项目带来的日益增长的风险。

事件起因于 MetaCubeX 组织近期更新的 Mihomo 项目授权协议。新协议在保持 GPL-3.0 开源协议基础上，明确要求任何非 MetaCubeX 官方的下游项目不得在名称中包含"mihomo"字样，并应避免使用可能被误认为是官方内核的任何名称。

然而，作为 Mihomo 内核的 GUI 下游项目 Mihomo Party 坚决违反协议，拒绝执行改名要求。据社区讨论显示，该项目已被原团队出售给某"机场"团队运营。

Mihomo 开发团队在公告中强调，遵守授权协议是下游项目应尽的义务而非可选的权利。团队表示，由于某下游坚决违反协议拒不改名，导致上游风评频繁受损，不得不考虑停止内核开发这一极端选择。

同时，Mihomo 团队建议相关机场赞助商及推广商认真考虑与违规项目的合作关系，评估其是否能带来正面的宣传作用。

Mihomo 团队在此前已表示，如下游不原意遵守协议，请对应项目自觉退回授权协议变更前的Mihomo版本（即v1.19.2以前的版本）

美国贸易代表办公室星期四（8月28日）在官网发布声明，通报上述消息。

印尼总统普拉博沃呼吁民众保持冷静，对骑手的死亡表达哀悼。他表示已下令彻查此事。当局证实已有7名警察被拘留和讯问，但司机身份尚未确定。
（美联社，路透社）

泰国被停职总理佩通坦8月21日上午在泰国宪法法院出席关于其涉嫌违宪的调查听证会。听证会上，佩通坦就其与柬埔寨参议院主席洪森通话引发的争议接受质询。宪法法院将于本月29日就佩通坦是否存在违宪行为作出裁决。 （新华社）

泰国宪法法院8月29日以6比3的多数意见裁定，被停职总理佩通坦在与柬埔寨参议院主席洪森通话事件中的有关做法严重违反泰国宪法对于内阁部长道德规范的规定，其总理职务终止自7月1日被宪法法院停职当天生效。
裁决公布后，佩通坦表示尊重裁决结果，但重申其初衷是为了国家利益，自己最重视的始终是人民的生命，希望今后各方能够团结一致，恢复政局稳定。
按照宪法规定，当总理职务终止，全体内阁成员也须一并卸任，国会下议院将负责选出泰国新一任总理。代理总理普坦将率内阁其余成员继续履行看守职责，直至新内阁就职。
分析认为，当前在国会下议院中仅占微弱多数的执政联盟缺乏突出人选继任总理，议会或将经历多轮投票方能选出新总理。
