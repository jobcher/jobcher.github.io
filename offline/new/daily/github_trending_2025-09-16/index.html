🖼 OpenAI 发布 GPT-5-Codex

OpenAI 发布 GPT-5-Codex

OpenAI 于 2025 年 9 月 15 日 宣布推出 GPT-5-Codex，这是在 GPT-5 基础上专门优化的编程模型，面向 Codex 产品线。该版本强调在真实软件工程场景中的表现，能够独立完成大型项目构建、功能扩展、调试、重构及代码审查，并在交互式会话中保持高效响应。

GPT-5-Codex 在 SWE-bench 全量 500 项任务上进行评测，展示了更强的推理和长期任务执行能力。官方称其可在复杂任务中持续工作数小时，迭代实现与修复错误。模型也强化了代码审查功能，被用于自动检测开源项目与 GitHub 提交中的潜在缺陷。

Codex 近期还更新了 CLI 工具与 IDE 插件，支持在终端与 VS Code 等环境中与模型协作，增加任务进度跟踪、截图输入、自动化环境配置等功能。Codex 目前包含在 ChatGPT Plus、Pro、Business、Edu 与 Enterprise 套餐中，API 接入计划稍后开放。

来源：OpenAI 官方公告
